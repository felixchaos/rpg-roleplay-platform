# 力量体系

## 薇瑟帝国骑士境界（从高到低）

| 境界 | 说明 |
|------|------|
| **烈锋** | 个人战力顶点，必须借助 aldnoal 能场助力达到；配合领长机甲是战场尖刀 |
| **渊戮极 / 渊戮巅峰 / 渊戮** | 精英层，帝国登记约700、实战老道约500；各贵族珍贵资源；渊戮极为最高子级 |
| **熔殇精英 / 熔殇巅峰 / 熔殇** | 中坚力量；熔殇无法在渊戮手里走过十招；熔殇巅峰已是强者 |
| **霜极精英 / 霜极** | 精英骑士层 |
| **顶王巅峰 / 顶王** | 进阶层；地球改造人普遍上限；界冢伊奈帆当前境界 |
| **永黯** | 基础改造人层 |

**注：** 骑士等级考试到熔殇级是官方证明；渊戮的民间标准是手下至少一百位熔殇骑士。

## 领长机甲

- 核心战争兵器，体积庞大
- **类型**：攻杀型、特殊型（兼具攻杀+特殊功能）、支援型等
- **无人子机**：约为领长机甲八分之一大小，假装多架领长机甲的欺敌战术（末章薛克用法）
- **光学隐形**：薇瑟高端军事技术，可隐藏领长机甲
- **战斗力公式**：渊戮骑士 + 巅峰领长机甲 = 单杀空舰

---

## aldnoal 特殊能力

- **感知善恶**：娅赛兰天赋，天然辨别人心真伪
- **境界传导**：蕾穆丽娜可通过肌肤接触将 aldnoal 流入他人（辅助斯雷因突破渊戮巅峰）
- **血源融合**：极危险的 aldnoal 生命技术，失败者转化为晶体
- **势场隧道**：机甲专属技术，长枪可在指向性能量场中瞬间加速（墨言东的马尔斯机甲）

---

## 关键规则

1. 反aldnoal体系只能从三维限制，源血脉者（蕾穆丽娜/娅赛兰）不受制约
2. 烈锋境界的达到需要"第一次攀登"后才能持续发挥
3. 各贵族渊戮分配有限，薛克一次性集结8位渊戮是极其罕见的举动（末章）
4. 境界跃升≠线性积累：斯雷因从熔殇到渊戮巅峰是意志蜕变+外部 aldnoal 助力的结果
