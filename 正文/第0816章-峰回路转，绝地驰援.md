# 第六零六章　峰回路转，绝地驰援

因为突然感觉共铲党不够防和谐。

【蕾穆丽娜】

然而，更远的天边，突然锐光飞起，流彩的能流横跨天际，纵出奔流汹涌的攻击，直接如箭雨枪林，在空中收束，对准飞来的四架精英机甲，投射而下！

精英机甲不是支援，而是撤军或逃兵！

这时，反倒是我笑了出来。

“真可惜，看来是我赢了呢。”

慕倪纣的脸色不太好看，但还是舒展开，他一挥手道：“撤。”

立刻几个女战士就围着她，跑向甲板边准备跳海……估计是准备走潜艇路子吧……

“我绝对会查明你的身份。”

在跳下前的瞬间，对面的慕倪纣看着我的眼睛冷冷道，一点也没有平日里的懒散和灰暗劲，目光中充斥着冰冷刺人的锐利。

“〖其它人！向各个方向分散支援救援！〗”薛克的声音自头顶上的领长机甲传来。

三个探照灯直指我。

“〖发现赛里朵亚郡主！〗”

“〖抱歉，我们营救晚了。〗”和薛克那些收拢的手下并不同，虽然有上面的调令，但基于赛里朵亚贵族的身份，我可是被他邀请过来的。

薛克要为赛里朵亚等几个这样贵族、骑士负责的，所以丢了一个赛里朵亚，估计也够他这几天愁的吧。

没有他在面对“蕾穆丽娜”时的侵略和猴急，这个时候的薛克就是众人理想而坚韧执着的队长。

“薛克队长，胜了？”

我仰头问道。

“〖大胜谢里曼丁！我等联合谢克尔、索里文、法恩泽共四支编队，大胜以特恩德、谢里曼丁、凯托文为首的地球联邦印度洋九支本海舰队构成的联合舰队！〗”

看来是大胜，这种事情都在这直接说出来。

“队长你来的真慢啊。”

感受着海面早晨的阳光扫在脸上，闭上眼睛我一时有些恍惚的暖洋洋，感觉这一路被俘虏十几天，就好像有一年般的漫长啊。

精神不是很足，好像随时都会昏睡过去。

“〖哈哈，队长你被姑娘抱怨来得慢了。〗”

“〖没办法，谁让我们薛克队长一表人才，花见花开，我去营救的舰上姑娘们都抱怨没见到长官呢。〗”

“〖哈，他们就算见到也是见到了队长的领长机甲，又不是本人。〗”

“〖姑娘们的花痴可不关心格你这个大老粗说的那些。〗”

薛克手一挥，随行的运输机便降低高度，亦有战士顺梯而下，为我检查状态。

“〖温茵－赛里朵亚，欢迎归队。令汝身陷伏围是吾等失职，然出卖情报的虎克－斐柯伦罪已伏法。〗”这算是薛克走的帝国贵族间的规则，毕竟他作为带队失职让我被俘，是要向我解释的——这算是贵族圈的特权。

我则摇摇头道：“没事，队长职责所在，赛里朵亚归队。”

……在外这么多天，爱乌得担心死了吧……

……

地球联邦标准时间，2014年5月26日。

印尼地区。

自24日我从地球的辅助舰上被救出来之后，我便被安排了两天的休息时间，用以缓解原本在被俘虏期间产生的疲劳和相应的身体负担状态。

其间也有军方安排的医生对我的身体进行检查，在对于我能够在那样困难和饥饿的情况依然战斗表示惊讶和不可思议。

虽然我对于被别的军医检查身体状态也有些紧张，然而毕竟血脉者带来的权限优势使得只要我想，对方在开动仪器的瞬间，什么异样都检查不出来也可能。

这也是为何之后的连续两天里我都在寝室里安睡的原因——相比于表面上检查出来的报告“仅仅是精神、身体过度疲劳”不同，通过aldnoal在体内反复的运转，我实际上一直处于精神与肉(谐)体的双重极限透支状态。

回到寝室之后我便直接倒在床上起不来了，长时间的紧张和谨备状态，带来的不仅仅是疲劳，还有意识与精神上的损耗。

躺在床上，睁开眼睛看着天花板的时候，整个人的大脑还是处于似乎停转的状态。

眼神看到的好像不是白色的天花板，而是曾经辅助舰上最后海夜里火光与血液四溅的战斗，还是手中拿着那柄战刀，一反往日里身体脆弱的状态，一下一下地将骑在身(谐)下穿着动力轻甲的男人的头骨砸碎砸烂的画面。

说是杀人，但实际上杀的感觉依然沉重。

“郡主。”身边早起已经准备好所有的爱乌看我没反应，便又叫 了一声，“郡主？起床啦，您从前天回来开始就一直躺在床上，总是像这样躺在床上对身体也不好啦。”

“啊。”我偏过头，看着依然简单女子军装长裤的爱乌应了一声，而后想要起来，却发现想要坐起来都有些困难。

啊，现在没有把aldnoah灌注到双腿中，再加上全身的骨头、肌肉都好像废了一般，我伸出被子的右手像小动物般上下摆着道：“起不来啦，拉我起来。”

“好的，好的，我的郡主。”说着爱乌走到床前，一手拉着我的右手，一手直接探到我的膝下，并不仅仅是把我拉起来，而是把我直接从床上抱起来，“在海舰上的条件不好，现在回到陆地的基地了，郡主应该没洗澡了吧？”

说着爱乌还凑过来靠到我颈间嗅了嗅。

我推了一下对方鼻子：“别嗅啦，刚回来时你急匆匆地把我泡水里洗了，再说我的体内有aldnoal体系保护一般不会脏的……“

“但衣服会脏的，还有自然排泄。”爱乌瞪了一下眼睛，而后笑着道，“反正这是我的职责所在，郡主这样自律会让我失职的。有机会也要慰劳一下希依呐，她可是那些天一直守在海上的。”

我点点头道：“当然，我当时虽然人在舰里，但一直知道她在我头顶上，如果不是她，我也不可能坚持下去。有没有人守护在身边时的底气毕竟是不一样的。”

爱乌掏出手机关掉了通信键，我一下子就明白刚才原来爱乌刚才对话是为希依求的呢。

毕竟和爱乌不同，希依因为驾驶迪斯派尔的原因，一直要守在外围，这对于她这种喜欢二十四小时盯着我的丫头而言，也算是考验了。

我坐在单人的小浴池里，身边的爱乌则穿着简衣坐在凳子上帮我洗头。

看着大腿靠近膝盖的位置那两道环绕了整个大腿的缝合疤痕，好像隐隐约约还有幻痛，我一时陷入了头脑的空白。

相比于之前头脑中总是塞满了事情不同，从海舰那里回来之后，我的头脑依然有些空白和迟缓……还是伤到头脑了么？

“郡主？”

“啊，没事，想事情而已。”

“哦。”

回来在军医那里重新检查了血液对瘾品的成瘾性，也不知道是不是aldnoal血脉者本身的优势，我对瘾品的成瘾性并没有形成，意识也相对稳定，也没有产生偏极、多疑、怀疑、冲动等过激反应。只是还是暂时有些反应空白和迟钝了不知道什么时候恢复。

虽然我那艘船上的俘虏除了我基本都死干净了……或者应该说是确实死干净了，然而因为是我队长的薛克亲自来救，所以即使只救出我一个人，其它的质询压力也在他那里被顶下来了。

因为薛克的队伍里，被抓到俘虏是相对最少的。

“爱乌，扎兹巴鲁姆体系军队用的战场药剂，有兴奋剂么？镇定剂都包括什么？”

我突然没头没脑地问道。

“噗。”爱乌听我突然问道，也一下子笑了出来，“郡主您是过度担心吧？放心吧，两星之间并不是所有的军队都在军队中使用‘瘾品’，扎兹巴鲁姆体系的军队中是禁止用瘾品、兴奋剂这些的。不仅我们井体系、叶体系、华汉的军队就我所知也是禁止这些的。

我们扎兹巴鲁姆体系本来就这是禁瘾严厉的体系领，自二代伯爵大人开始，带领的领系团队就一直是禁瘾的，制定的法规也是学习华汉体系的严厉，贩瘾者直接死刑。”

“嗯，其实我虽然当时尽量少吃了，但现在看来，还是有时会反应感觉跟不上……还是有些伤了。”我叹了口气，“要是没瘾品那东西，也许舰上的那些人不会那样吧……”

爱乌则用温水冲过我的头发：“郡主的事情我昨天和格列维尔斯谈了，他说，在那样的分割审讯和攻心还有有意的各种假情报操作下，即使没有使用瘾品，大概俘虏们最后也会陷入分裂和疯狂，瘾品只不过是加快这个进度和过程。“

攻心为上么……毕竟格列维尔斯和马克－贝伯爵他们是搞情报这一块的。

“你只说了我们领还有井领、叶领、华汉，其它 的呢？”

因为过去培养和成长的原因，我也很清楚自己的定位，只要扎兹巴鲁姆伯爵没和我谈，我从来不去主动地接触军队方面的事情。

大概也不仅仅是我自己定位的原因吧，面对祖祖父那强烈的单谈命令，我本能地拖延自己接手扎兹巴鲁姆领军队的事情。

“当然是那些军队内流通和使用瘾品是合法的啊，他们被认为是镇痛剂、安慰剂、兴奋剂、缓解剂。毕竟我们这边的思想政治教育缓解不同，某些领和地球的洲区军队主张更相信一针下去对战争中紧张的士兵的缓解作用。”

而后爱乌也撇嘴笑了笑：“至于之后的成瘾或是副作用，也许在那些大人物眼里，士兵和我这样的类似存在也都是消耗的数字吧。”

如北美、欧洲、细斯米塞等等那样的军队，瘾品是军队内配量合法的啊……
