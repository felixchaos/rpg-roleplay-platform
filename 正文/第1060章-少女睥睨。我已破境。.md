# 第二三五章　少女睥睨。我已破境。

看到连“埃德尔”这样大人物都会出现在小小医院之中，雷满多眼神震惊无比，也立刻明白过来之前伊奈帆和莱艾的表现因由——既然是能够作为文木奥姐妹的护卫，即使只是顶王水平，那也应该是顶王之中的佼佼者和资优者，与他这样的劣质者自然不同，表现出来的作战效能也是不同。

雷满多并没有因此而自卑或嫉妒，相反能够在欧洲世家强权面前叫来真正愿意主持公义的人，雷满多心生感激，并且已经看到当前的事情终于是有了一个定局。

他不仅要自己记住这个恩情，也要把这件事告诉尚且在病房中精神不清醒的袁帅。

只不过双方的差别实在是有些大……一方是可以接触到星际政局顶层的少年精英护卫，一方只不过是华汉军队中的小小改造人，可能这次过后就再没见面机会了吧？

雷满多的想法只是普通人的惊讶和心中不可置信以及意外的惊喜和解脱，但对从头到尾都超出事态掌控的尼特鲁而言，今天一天从伊奈帆出现开始的感受就只能用吃屎、吃苍蝇、吃垃圾来形容了。

即使他根本没吃过上面的东西。

尼特鲁本来还冲动地想对埃德尔利佐喷回去，却被沃尔洛德随行护卫和跟班马仔少爷们劝住——

尤其是在文木奥姐妹中的妹妹曾经有过被刺杀，而这个带伤绷带的男孩更是在会场为姐姐挡下三枪的少年英雄，他们这边完全不占任何的优势了。

“哼。”

对方不敢接埃德尔利佐的问话，对方也不能接埃德尔利佐的问话。埃德尔利佐敢于如此自信除了她身上的威势，她们姐妹二人在欧洲做出的一系列事情和政治存在感，亦是因为慈善机构本身就市值二百三十亿欧元这样的巨资庞物。

更何况真接了这话，往小了说当然可以化解，但往大了说却是可以上升到外交问题上——她所谓的姐姐实质就是公主殿下，借点虎皮拉大旗。

真的让一个联邦的世家臭下去出问题，她作为薇瑟人可是乐见其成的。

以娅赛兰的性格确实会因为“爱与和平”原谅对方曾经对袁帅与伊芙的刁难，但没有关系——地球联邦的欧洲洲区军队军用兴奋剂失败品却被用出了性chūn药的效果，并且这些“青年军官”们还借此以“迷jiān”，即使实际上他们是靠着家势对那些姑娘们予取予夺。

但没关系，过去埃德尔利佐对韵子的调(谐)教或者说严格在慈善机构里训练出来的成果有了充足体现。当伊奈帆叫埃德尔利佐时，韵子亦花了很短的时间把袁帅伊芙以及尼特鲁的资料调集清楚——尼特鲁的黑材料真的不用太麻烦找。

十四岁的女孩在这里最矮，但她站在这里，就悠然地一步步走来，越过伊奈帆，越过莱艾和雷满多，走到了大概与尼特鲁和对面沃尔洛德十步远的地方对视，却从头都没有人敢质言。

“哼。”埃德尔利佐用鼻子又讽笑了一声，“内斗内行，外斗外行，这可是你们联邦的俚语呢。或者我用个联邦历史上挺耻辱的名言怎么说？也许南面的薛克侯爵要说“十四万人齐卸甲，竟无一人是男儿”了——听闻乌里克欧兹侯爵最近刚刚兵不血刃地从内策反拿下了两洲交界的河道口啊。”

这话的潜在意思自然无比辛辣——南面对面就是比诸位青年远要年轻的十九岁侯爵，他都已经在战场上领军建功立业了，诸位却是废物纨绔人生。

埃德尔利佐上下打量对面的尼特鲁，看着对方的脸色从最初有些不快的微红变成憋怒的闷红。但搀扶穿着病服埃德尔利佐的韵子手里就拿着手机，开启着录像录音，她埃德尔利佐可等着对方有谁敢接话呢——没人敢怀疑火地慈善机构能够对外传播的媒体影响力。

至少要钱的话慈善机构并不缺钱，过去她们名义上的总裁兰赛欧莉小姐往外扔了多少钱，数得清的人惊讶，数不清的人只知道对普通人是天文数字。

“唉……这世家，也就，不过如此罢了。”看对方从头到尾也没敢接埃德尔利佐的话，埃德尔利佐叹气地摇摇头。

至于世家虽然没点名，但埃德尔利佐讽刺的路易威斯家，听的人都清楚。

尼特鲁早就忍无可忍想要谩骂埃德尔利佐，但却被沃尔洛德直接扣住，让他说不出来。

眼前的“埃德尔－文木奥”小姐有多么得伶牙利齿，言辞尖锐，虽然已经是收敛许多了。要知道他之前看新闻上这位小姐可不像她的姐姐那么亲善于地球，什么样的不带脏字却能字字要害的指责地球，她随口能来还能落地有据。

埃德尔利佐将拿在左手中如手机大小的电子板甩给对面，冷言道：“自己走吧，还要我送你们么？不怕事多？难得住了病院，我可是闲得很。”

尼特鲁的跟班接了电子板一看上面全是尼特鲁的黑材料，之所以如此详尽不是因为慈善机构的情报部门效率高到爆炸，而是因为慈善机构总会接触到各个欧洲世家，那么提前准备各种材料是没错的，今天自然就用上了。

“二爷，我们走吧……”跟班和沃尔洛德都劝道。

但尼特鲁却极度地不甘心，被前后两个人这样羞辱扫面子最后还要灰溜溜地走？他的面子全扫地了。

但他可能根本不清楚，正是因为他这样的人，才会看重这种面子，他还在担心别人以后会借此事嘲笑他？实际上无论伊奈帆还埃德尔利佐都只会将他当作过眼的昨日之尘，根本不会再放在心上。

也可能，便是他这样的假橘子，和眼前埃德尔利佐这般源内而外，生而贵族的华雅高然的真橘子不同，也是尼特鲁眼红的原因所在吧……

故而在没有人可以针对时，尼特鲁环目四顾，突然发现最后依然只剩下伊芙了。他像落水疯狗般叫道：“不行！卡夫特奇，你要把伊芙给我抓回来！一切都是那个贱人的错！是她引起的！”

马上有马仔和顶王队员要去病房抓人，但这时却有熟悉的身影冲来，拦在了袁帅病房前。

“呼呼……许久不见，尼特鲁－路易威斯。伊芙她不能跟你走，就算她是之前我母亲当物品送给路易威斯家的，这样的约定现在也不生效了，因为我成功给她办了入华汉籍申请。”

莱德看来赶来得很急，平日里贵公子英姿飒爽的他也有些汗水，可他还是笑了出来：“很可惜，按照华汉宪法，任何的华汉公民都不会允许受到非法的奴隶对待，并且有资格受到国家的保护。”

听到这样的结果，雷满多和伊奈帆等人都很意外，没想到这位世家少爷是真的把自己的一个小侍女当妹妹看待，为她做了许多的努力。华汉虽然当下正处战事，但华汉的国籍却一直都是世界最难拿到的国籍之一。

想要这么快办下来，莱德一定和华汉的组织上申请过，并且付出了某些代价，只愿兰佩路基夫人不会因此和她的儿子生气吧。

但尼特鲁听完却大笑了出来，他指着莱德的鼻子大叫道：“什么！？你给那个贱人办了华汉籍！可笑！太可笑了！哈哈……这什么鬼！你不知道你养的白菜已经被你手下的猪拱了么！？还在这里给她办国籍，是想直接方便她们结婚么？

从头到尾只有你这个绿帽傻子什么也不知道，自己上过的女人回头又让自己的手下上了，哈哈哈哈……“

尼特鲁终于在许多不快之后又得到了一份难得的快乐，他放肆地在医院大笑出来。

但莱德却一脸懵逼，他瞪大了眼睛回头看了一下身后的病房，心里一下复杂不已，但面上表情却勉强镇定——要是秦月弯知道他没帮她看好袁帅，还不知道怎么找他麻烦。

“她和他好上了？上了？”莱德说得有些愣了，“那……恭喜啊……”

雷满多站出来解释道：“并不是她们交往了，只是因为袁帅代伊芙喝那个药酒，没想到后劲过大出问题，伊芙才主动发生关系……”

这话说得雷满多也有些尴尬。

但莱德听完却由原本的表情愣然转为有些严肃：“这小子，做事还是这样，又不知道给自己收底，也不想想后果会不会太严重……要是境界受损怎么办……”

“哈哈，你小子总算是认识到自己被绿了？终于严肃了？放心，我又不会为难你，之前从令堂那拿到伊芙时我都没碰，就怕你麻烦，哈哈……”尼特鲁大笑着莱德，“你个蠢货，养了那么多年的花让野狗叼了……哈哈……”

这样放肆而没有教养的声音让埃德尔利佐和莱艾皱眉，只有伊奈帆依然面无表情地不在乎。

但莱德终于是把目光转而看向尼特鲁，像是第一次正视一个“原本被无视的东西”，他顿了顿道：“我没碰过伊芙啊，我一直当她是妹妹，她这之前都是处(谐)子。“

“嘎！“听到莱德说法，一下子笑得岔气的尼特鲁脸色僵硬。

“我和你没完！别以为申请成功就有用！证还没下来，在欧洲这我有的是法子弄死她！弄得不得好死！卡夫特奇你找机会就去办！“

莱德则眯起眼睛，看了一眼尼特鲁之后看向对面沃尔洛德，神情严肃道：“希望您认真审慎，因为如果您真的想如此而为，即使我不代表兰佩路基家，但我依然代表我自己，您对沃尔洛德之下可以碾压，对我没用。“

“凭什么对你没用！？莱德你小子也不过是顶王！一个大男人难不成也请个沃尔洛德护卫！？“尼特鲁叫嚣道。

“这次回来，我已经破境入沃尔洛德了，我能给伊芙申请佳能国籍，代价便是我的沃尔洛德。”莱德声音平淡，却又认真道，“欧洲，不适合她这样纯粹的孩子。”

“……你破境了！？不可能……这绝对不可能！你都失败两次还被记了军队处分！……你怎么可能成功！”
