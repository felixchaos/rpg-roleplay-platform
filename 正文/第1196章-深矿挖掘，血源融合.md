# 第四一一章　深矿挖掘，血源融合

【蕾穆丽娜】

显然斯雷因也清楚“外行谈战略，内行谈后勤”，战争即使有再好的战斗意志，也是需要最基础的后勤来支撑的，而余下的差距就是需要人命、战术战略和战斗意志等多方面来填补。

“上个月空舰、机甲制造速度和战场消耗速度持平，弹药消耗与生产持平则在上个星期，如果战争继续升级烈度，那么实际上华汉的武器与弹药产能还会继续向上跃升，和消耗保持一个相对平衡。”

斯雷因听我讲到这里，也自己用手指在地板上画了两笔，大概是在通过最近所学习到的知识来估算华汉与皇派轨道骑士之间的产能等多方面情况。对于我为什么得到这方面数据斯雷因不疑有他，显然相信我和扎兹巴鲁姆领有这个实力。

然而实际上，这些看似粗略但在实际战争中却同样都应该保密的信息却是以原封的形式呈现在了扎兹巴鲁姆伯父的手中，进而再通过最近的情报网络因为传递而时间滞后来到了我的手上。

究其原因，还是因为艾文当初和华汉签的那些协议带来的效果——虽然艾文回来就被监(谐)禁了，可实际上大家都不是小孩子。如果协议有问题那么随时都会被伯父他们撕毁成废纸，艾文当初的独行也可以被推出来——毕竟只是以个人名义签的，这份协议是否真的有执行效力还是要看两方面：一伯父等人构成的扎兹巴鲁姆体系领导团队，二是我这个未来扎兹巴鲁姆体系继承人。

伯父身处领导顶点，自然秉公不可能有所偏帮，所以艾文的释放令和最后签字是我。

当然反过来既然协议还在，那么这些“无伤大雅”的数字如果扎兹巴鲁姆体系需要华汉也会提供，作为双方进一步合作的证明。

当前战局发展也正如艾文和我一开始所想，因为没有在第一时间完全地吃掉华汉所以随着华汉工业体系逐渐地启动，反而是在远场作战的皇派有些撑不下去的样子——人口、动员、资源、组织，这些方面华汉都不会比皇派差，或者说由于大量人口带来的巨大动员力量，华汉反而在这方面有所优势。

aldnoal确实是一个巨大的产能宝库，它使得本是半封建半资本的薇瑟帝国可以产生堪比资本主义、社会主义国家混合地球联邦能够爆发出来的工业水平，甚至更胜一筹。然而这些也就是如此了，因为阶级与贵族等多方面的固化，在内部的资源与调用流通上，薇瑟不会做得更好——除非它们能够革自己的命来进行改变。

但这本就是不可能的事情。

“当然，战争打到现在，实际制约的还是人，就算可以按照图纸生产出来更多武器与设备，然而如果没有足够的兵源与人口去支撑，那么自然也是空谈。”

“可是皇派不是占领了华汉东部的沿海，原本华汉20亿的人口现在有将近6亿……”斯雷因说到一半自己反应过来，苦笑了一下：“可惜贵族们并相信占领区的华汉人，现在对占领区的华汉人所持的态度大概也还是放养和不信任吧？”

“当然啊，本来就是在战争时期又因为有军官在华汉的市区有过遇害和袭击，贵族们不可能把军队的管控交给华汉人，这些占领人口反而是放养的累赘。而如果制造些无聊的伪军权限上也不可能实现。”我耸耸肩回应道。

帝国的军事设备都是有权限锁定等涉及aldnoal操作的。如果要打造出一批由华汉人组成的伪军那么就首先要授予华汉人权限，而这对于当下国内政治敏感的局势并不是什么容易被既得权限众人所接受。

薛克在印尼地区对政治、社会、平民地位的改革，甚至是对部分人群授予了以乌里克欧兹体系为下辖的权限，都在国内受到了很大的微辞。这也是为什么他近期的时间就完全是留在了非洲－欧洲的前线而根本没有回过本星。

“产能能够跟得上，那么资源呢？”斯雷因又问了一个涉及到基础方面的问题，“尤其是矿产，当下战争时期能够获得的矿产不可能如和平时期那样的通畅，这个方面他们是怎么解决的？”

“一部分是从俄北获得的，另一部分是应用了深矿挖掘技术。”

“深矿挖掘？”看来这是一个斯雷因没听过的名词，他眼睛转了转似是在回忆起是否特洛耶德家的技术遗产上有过这方面的内容。

“啪啪。”我拍了拍手把斯雷因的思路拉了回来，笑道：“不用想了，深矿挖掘不是特洛耶德家的技术，而是扎兹巴鲁姆近年才开发出来的。主要是向比之常规挖掘更深层面的地层深处高速高效挖掘。因为aldnoal存在矿产问题对帝国从来不是什么问题，所以薇瑟并没有这方面的技术，如果真的需要也可以动用领长机甲对地壳进行粗暴地打穿，然后再提取矿产。“

斯雷因听完理解了一下，然后看向我道：“但实际上，开发这项技术本来所应用的并不是挖掘地壳深处的矿产，而是为南极的古老原始的生物冰封层所准备的？”

“聪明，没想到你能想到这个方面。”我看着斯雷因的有些犹豫的表情，抬了一下眉毛示意道，“但其实还有没说完的吧？”

斯雷因叹口气，看了眼窗外几秒，大概他在想些什么事情吧，然后他才看向我：“蕾薇你……早知道两星要发生战争？深矿挖掘，如果是要用于挖掘南极生物冰封样本，那么就不可能如正常的挖矿那样粗暴，必然需要更高的技术水平。这样一项技术和工程组合还有后方工业的动员准备……都是需要提前进行规划的……

也就是说，蕾薇你不仅在大概五年前就知道两星之间要发生战争，而且也知道……这场战争持续的时间不会如过去那样超过三年，而是大概会在二到三年内就结束？“

斯雷因这时看向我的目光无奈又复杂。

我愣了一下，才反应过来斯雷因会瞬间想到这么多。如果我知道的，或者能预知的东西有那么多，那么娅赛兰相应是否会被刺杀的事情？我在这一系列的事件里的立场就非常得微妙了。

——有可能是背后的掺手之一，有可能是冷眼旁观者。

而我却没有想到斯雷因能从一件信息里去串连出那么多的内容，也对啊，无论是斯雷因还是别人，又有谁是傻子呢？

更何况斯雷因对事情的分析能力还不及艾文，就像下棋我在艾文手里都不可能赢，我能够预判的除了前生的记忆就是因为前生的知识让我可以暂时地跳出自己所处的立场，从另外一个角度观察问题。

“愿意当面说出来的信任，我很感谢。”我似笑非笑地答道：“要说五年前，我也有过这样的判断，两星之间的战争可能会在近年之间发生。

对于娅赛兰被刺杀的事情我很抱歉，并非是参与，但确实是冷眼旁观——她想来地球本来就是很早就有的想法，而她一旦来地球人中的激劲派就就不会放过她——不管这个刺杀的背后有没有薇瑟人想以此为战争的导火索。

至于说战争打不过超过二三年，那是因为当前人类技术水平用于战争说一旦烈度达到顶点，那么人类自己的人口就不会撑过三到四年的消耗，而双方必然不可能把全部的人口都投入进去，把国民打空，那么三年自然也就是上限了。“

“确实……抱歉，知道并不意味着要阻止。是权利越大责任越大而非能力越大责任越大。如果只是我自己的思考，确实能考虑的问题的方面和远度都不够。”斯雷因听了之后不疑有他，而是歉意地向我点头，“更何况你是对扎兹巴鲁姆体系的人民负责，而非华汉或世界两星。”

“无妨，保持思考反而是好事，如果只是因为我们之间的合作就完全的信任，那么你将来与他人合作也可能被背叛和背刺呢。”我用食指的指尖轻轻敲了一下手边的电子板屏幕，而后道，“那么我继续说下去了？”

“嗯。”

其实，如果斯雷因能够再往深一点想，如果斯雷因的眼界能够再看宽一点大概就能够看到更多的内容，不过那本来就是不现实的事情。

如果真的能够看到那样层次的高度，那么前生的斯雷因也不会沦落到孤岛监(谐)禁的终局。

如果不是因为我自己就身处其中，连我自己也不会想到。而如果他会想到，那么他对我个感官会完全逆转吧……

而每当看到眼前世界的境况时，我却又不得不感慨如果真的论思想与认识的高度，以马克思等一批先贤的眼界高度依然值得后人学习。

可那又如何？今生的火星是在薇瑟贵族和皇室的控制之下，他们并非推崇哲学，而是更多的地迷信于aldnoal文明所带来的超越时代的科学、技术和能力的伟力之下。

“我们知道在人体培养方面，薇瑟的骑士质量上都要超出联邦的改造人，虽然两方各有自己的人体培养评判等级，但熔殇和顶王之间实质也没有太多差别，真正的差别还是最后培养出来的战士的质量。

缺少了aldnoal，改造人必然要比骑士差这是一个不争的事实。因为知道缺陷，联邦也不可能真的就放着不足的地方无视而不去想办法改变。我今天得到的消息就是华汉的神经激生药物实验所与俄北的沃尔洛德破境研究所取得了血源融合技术实质性的突破。“

“血源融合？”
