# 第四二九章　天地人心，混乱众生

【无】

当然，如果他的想法为蕾穆丽娜郡主或是波塞罗斯伯爵知道，那一定会被直接臭骂他没有一点姿势水平。

嗯，以蕾穆丽娜郡主的性格，想来更可能的是直接无视掉这样如臭虫般寄生在华汉人身上又为害华汉同胞的垃圾，看一眼都是浪费原本应该倾注在她负责的领民身上的关注。

实际上，人类即使走到了今天的科技水平，走到了能场覆盖的角落，就是帝国或联邦的领土，但人类依然没有跳出作为基因载体的桎梏。

骑墙派、妥协派、投降派，这些所有在同星人同国人同民族同胞之间看来明显是卖星、卖国、败类、叛徒的同胞，实际上也都是受到了背后那浩如烟海又繁杂多变 的种群基因本能诱导。

当然，这并不是说，他们的罪恶和出卖同胞，为侵略者带路为害同胞的行为就值得同情、理解和开脱——本能和基因是一回事，人类的思想、道德、民族、公理、律法又是一回事。

这些同胞血脉垃圾本就是该处理的对象，而实际上，它们都只是受到了种群那为了延续而做出的多方投注赌博行为而已。

就像某个岛国、某个欧洲国家、某个华汉历史的封建王朝、某个火星帝国过往皇帝曾经做出的赌国运一样，大到国家，小到个人，赌博、投机这些完全看运气，没有一点理性成粉的因子，一直在基因的刺激和诱导下驱赶着成群的人类走在一条麦杆架在峡谷两边的独木桥上——

将同胞挤下万丈深渊的下一刻，可能这根麦杆会断掉，而后自己也掉进去万劫不复。百不存一的赌注，依然会有无数的赌徒忘记了理性蜂拥而至。

人类改不了的天性。

实际上，薇瑟军队内的倾轧、联邦军队内的出卖等等，都只是这样的本质所浮现出来的表象。

而人类的理性和定立的规则，与其说是是约束人类，不如说就是约束着这些胡作非为的基因的枷锁。

人作为理性和本能的矛盾集合体，作为思想与冲动的交织第一线战场，这条斗争线就从来没有停止过。

或者脱掉肉(谐)体的桎梏，走上思维理性如神的巅峰，升华到与宇宙同存；或忘掉思想，与本能和身体延展出如科幻虫族的本能吞噬毁灭一样的灾难族群，直到消耗干整个宇宙的最后一滴能量，直到最后这个由“原人类”形成的怪兽因为宇宙这个“囚徒密室”“饿死”自己。

像这样深刻的社会学、人类学、进化学、哲学问题，早就在两星的学术界和泰斗之间争论过无数遍。科技有星界和国界，科学的理念却是没有国界的。

然而这样的争论到现在依然没有什么结果，而对于波塞罗斯伯爵而言，他也没有这样的闲心去思考那些至少以千年为单位的人类进化大事。他所关心的，是薇瑟帝国人民十年之内的未来——

他必须要给某些自作聪明实际上太年轻，想得太简单，思维太幼稚的人传授一些入手经验，提高一下姿势水平——

否则，等到历史的车轮滚动到谁都停不下来的时候，一切都晚了。

可惜，他的实际影响力已经让他力有未逮。

更为令他心焦的是，他怎么也想不明白，明明身居比他高位的人，为什么还是没有提升足够高的姿势水平？

也许，是因为立场、先入为主的观念和视野吧？

坐在专机上的棕色短发掺满白发的四十九岁老男人，突然发现……也许，那两个人，根本不会接受吧？

如果能请动同为逆派的索拉克尔达侯爵站在自己这边，那么，也能少一些华汉人的冤魂。可惜，信息就像石沉大海一样，那位年轻的侯爵大人，那位让波塞罗斯伯爵亦需要忘记对方的年轻，平辈相待的存在，却一直都没有表现出什么立场和期待。

而阿克蒙德伯爵作为一位更为正统的骑士、军人，则对此事爱莫能助袖手旁观。

波塞罗斯伯爵到最后都没有在马里诺登陆城见到蕾穆丽娜郡主一面。如果他还能知道蕾穆丽娜－贝伯爵这条线的关系，那么转道也许不会扑空——无论是交流政治看法还是拉拢在这件事情上的立场相助，都应该会成为一份巨大的助力。

可惜，蕾穆丽娜郡主化身的赛里朵亚郡主避而不见，因为背后还有许多的贵族关系，赛里朵亚郡主实际上没有几个人能知道她的行踪。

蕾穆丽娜曾经还以为会是一次没有了特权的地球之行，然而作为扎兹巴鲁姆体系的正统继承人，这样的想法实际表现出来的，连她自己也不得不承认，前生二十多年加上今生十年，已经三十多年的自己，相比于伯父和马克－贝伯爵等人还是太年轻。

“我不能见波塞罗斯伯爵。”

站在贝伯爵登陆城高空边台上，晚风皆被能场滤掉，一身军装长袖外套中裙黑长靴军服的棕发少女看着天边的月色。在aldnoal的无形透射和传递之下，波塞罗斯伯爵、蓝尔赛伯爵、库鲁特欧伯爵究竟 在干什么，她虽然不能完全知晓每一个细节，但大体的军事调动风会带着aldnoal告诉她。

寒夜滤掉了少女白日间最后的温情，只剩下面对这片星空，这片天地的冷漠。

她脚下的土地，是一个名为“华汉”而非“中华”的大地，相似，却又完全不同。

很多时候，她是没有资格选择立场的，任何人都一样。她从一开始就被打了上一半扎兹巴鲁姆一半薇瑟皇室，一半血脉一半私生子，一半郡主一半庶公主的标签。

“郡主这次依然去前线么？”

走到了身边同样看着夜色的马克－贝伯爵则手里还拿着电子板处理公务，相比于少女的血脉能力，他依然是普通人，自然没有什么拿眼睛看aldnoal的能力。

实际上只要蕾穆丽娜有想法，马克－贝伯爵自然还会安排人手，这都是可以作为郡主成长的一环。

没有坐轮椅而是靠着aldnoal站立的少女郡主则摇摇头：“人命，不是这么浪费的，这次离远的点的观察哨点吧。完全不出席军事行动还是太明显了。”

“郡主明明有过类似猜想，却并不与波塞罗斯伯爵见面，现在他自己势单力孤，势必不能成功。”马克－贝顿了顿而后道，“蓝尔赛伯爵和库鲁特欧伯爵未来的每一次行动，都可能会进一步加深地球人对薇瑟帝国的仇恨……”

马克－贝伯爵的话没有说完，而内容却不言自明：

无论是艾瑟依拉姆公主登基还是蕾穆丽娜郡主登基，未来的华汉的仇恨都不是说消就消的。

敛眉看着周边下空数百米的高度和下面渺小的山林，远处渺小的城镇，少女摇摇头，声音清冷：“我想我不会登基，而是继续在扎兹巴鲁姆体系里一直到老，除非艾瑟依拉姆连登基都做不到，老皇帝才可能允许我染指玉座。”

实际上，病重和虚弱交织的雷利加利亚一年中真正以身体坐在玉座上的时间可能都不够两只手，但那又如何？只要aldnoal的触手一天还遍布近乎帝国和贵族的每一次角落，他的意志就可以传达到帝国的每一个角落。

如果不是各个实权贵族和有能力的贵族都有自己的防aldnoal透入的密室、小基地，这些个帝国皇室心知肚明又睁一只眼闭一只眼的存在，贵族们就真的是完全如狗了。

虽然现在许多贵族看来，他们自己已经是皇帝手中的狗，故而有的狗想要挣脱狗链，有的狗想要换条新狗链，有的狗则想要咬死其它的狗等等不一而足……

实际上也只有扎兹巴鲁姆家族敢明目张胆的在大片的里尔塞军事研究基地都布署隔绝aldnoal的设备仪器，直到蕾穆丽娜的出现才对aldnoal开放。

可惜，进入那里的是蕾穆丽娜的触手，而非老皇帝的意识触手。

“也是。”马克－贝伯爵不置可否的点点头，而后道，“薛克世子一路周转周游世界，终于也是到了华汉。虽然不太可能会暴(谐)露身份，但还是请小心。”

“嗯。”

蕾穆丽娜点点头，未有接话。

“按照参谋部的分析，波塞罗斯伯爵很有可能会更快的离开华汉——当然，是被人礼仪的‘赶’出去吧。”说着，这个在夜色下依然深紫发与肌黄的皮肤，削瘦的身材而有些如鬼的男人一笑起来就如骷髅般颤动，他的声音沙哑也如奇幻故事。

唯我云霄伯爵已经离开了华汉洲区，前半夜登陆城升空，看这样子，蓝尔赛伯爵和库鲁特欧伯爵早就将华汉视为鲜美的半遮半lù的处(谐)子，但实际上真的是那样么？

也许，掀开对方裙子，捅向自己的兵器，比自己还要粗要硬吧？

尤其是在蓝尔赛伯爵自断臂膀般的和库鲁特欧伯爵赶走了波塞罗斯伯爵这位华汉通老司机之后。
