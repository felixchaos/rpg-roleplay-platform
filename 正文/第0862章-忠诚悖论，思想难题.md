# 第十七章　忠诚悖论，思想难题

【蕾穆丽娜】

当我这边已经与名为黎骨月，这个斯雷因唯一剩下的的侍女对话完毕时，斯雷因恰好也在刚刚被他所杀的男人尸体前伫立了缓缓几秒。之后白发少年又走到那位死于队友推刀的男人尸体前，微微的鞠躬以示歉意。

这在我看来是很没有意义的行为，但转而又一想……也许正是因为这样总是想要在事事之中做得尽量妥善一点的少年，才会是那样在前生可以吸引无数观众的特质之一吧？

和站在高位总是呼吁着其它人去做着她口中的理想……也许和她心中的理想也微有所不同，斯雷因确实在从他能力上做事。

“你……刚才其实也想杀她。”斯雷因迟疑了一瞬，还是没有隐瞒什么想法的挑明道。

我一时忍着身体不适和精神虚弱道：“我本以为你还会像之前卡兰德丽死时那样不忍得无视过去。”

“并不是不忍，也不是害怕，只是我想，如果真的要沟通，要表达什么，还是把事情都诚恳的说明白一些，也许能少些矛盾，少些猜忌和想法。”

少年抬起一只手解释道，“其实，刚才那个瞬间，我以为死的会是我……也许，是境界突然之间的复原，也许是意外的身体偶然适应与爆发。……虽然现在又回到没有境界的状态了……”

斯雷因说的时候回头看了一那个男人尸体一眼。

我看了一眼作为被判别对象，被谈论着生死的女孩女孩，而后向斯雷因道：“既然你想表达你的看法，那我也可以明说：斯雷因，我现在要想办法带你出去，那么接下来我们将会遇到事情，或者说秘密就不是信任不够的人能够随便跟随的——“

“咳咳，“我又因为气息不顺咳嗽了几声，而后抬手期擦了一下嘴角点点血迹继续道，“特洛耶德家族的遗产、整个地球之行、以及接下来路线，你想过这些东西如果在半路上就与一个并不能完全给予信任的人同行的问题程度么？”

顿了顿，我道：“就像你之前在卡兰德丽那样总想着两全其美，也许下一次的选择就是特洛耶德家族遗产和她之间——虽然那很不恰当，或者换个说法——娅赛兰的立场和她的身份不明所带来的危险，谁来担这个风险？你么？斯雷因。”

这本来就是一道苛责性质的二选一问题，或者说在哲学上这种类似二选一为难一般人，却又是一般人可能一生都碰不到或只碰到几次的问题，我却一直在用此类问题刁难斯雷因。

因为，他所处位置，决定他要面对所有人刁难。

因为，我不刁难他，自然还会有世界刁难他。

“我来担这个风险……”斯雷因停顿了一下，而后直视我的双瞳苦笑道，“是说得有些轻率和不成熟，何况，这种情况下我相信蕾穆丽娜你会做出相应的决策——即使是证据确凿对她的死，我也不会反对——当然，我同样愿意相信她。”

而后，少年转而看向黎骨月，有些无奈的歉意笑道：“很抱歉，我愿意给予你信任，可是她并不愿意。我依然希望，我们的信任能够延续成整体而不是只留下一个让人惋惜的结果。我相信你能够以自己将来的行动践行以证明，证明给她，也证明给自己。”

少年的面容和表情纯净，没有丝毫的个人私欲，那样为他人着想也恰好能够博得他人的好感——如果娅赛兰是鲜明的光，那么斯雷因就是纯净的水。

“不会令您失望的。”

黎骨月点点头。

我一手扶着墙壁，一边向前走去，同时并没有停下对黎骨月的质疑和意图排除——因为接下来一路之行，可能将会展现在几个人面前的不单单是特洛耶德家族的遗产，还有一个问题——

就是我自己的血脉者问题。

这也是我为什么要在黎骨月的手腕上套一圈aldnoal能场导通材料做成死咒的原因。但我同样也会担心，任何谍报组织不乏勇死之辈，黎骨月如果是拼着死的代价在接下来的行动中把这件事情以生命传出去……

“斯雷因，你知道么，所谓的忠诚，就是这一个矛盾的东西——它不可能通过掠夺获得。因为当背叛产生的那一瞬间，所谓的忠诚就成了一个笑话。不管之后做过什么——这就是为什么类似的爱情的忠诚，组织的忠诚，行动的忠诚，归属的忠诚是不可能被掠夺或改变的原因——

改变的一瞬间，所谓的新忠诚，已经是生于旧的反叛之上——

只是虚假。“

我在说的什么，斯雷因自然听得明白，既然有些未来的内容我已经讲明，那么他应该也会想明白。

“蕾穆丽娜，我觉得……这样的命题，有一个最大的前提，就是人的思想是固定的，不会改变——

诚然，对于三观已经形成，对于认识世界方式方法已经定形的人而言，那些所谓的表面改变可能并不是真实的改变，因为忠诚无法被掠夺。”

斯雷因缓缓吸了一口气继续道：“但人其实又是变化的，这个世界最奇妙的就是人的想法千奇百怪……啊，也许就像你看我这样——那么，总会有那样的一些人，他们会因为过去的经历而产生了很大的触动，想要改变——

那么，当一个人的思想都已经改变的时候，他还是原来那个忠诚旧对象的人么？他的新的思想，是否可能成为新的忠诚诞生的土壤？“

斯雷因笑容淳真：

“忠诚与信赖无法被掠夺，但我相信可以培养和形成。”

“哼……随你。”我冷嘲一声，“现在连黎骨月的想法你都没有完全搞明白，却想着要可信，斯雷因你也是异想天开……咳。”我踉跄了几步，而后没有站稳，侧靠在墙上。

很清楚听到身后斯雷因一时急步想要过来，但他又停下脚步。

他低头看着自己握紧拳头道：“但无论是信任关系还是看法，都会有一个形成的过程，我想……我想，这总是要有人来做。也许，蕾穆丽娜你一直对娅赛兰嗤之以鼻的原因，便是……她一直在用她有限的方法去做着她自己的努力，但那样的努力，在身处众实权贵族的环绕之中，依然是无力的。”

白发少年又把拳头用力张开：“但……就像从理论到技术，从想法到实践，总是有一个过程。也许我的想法、愿景都是源自于娅赛兰，也许这样的想法总是被人说是天真——它也许真的很天真——但，想要把它们实现，总是要有人来做的……我想成为那个第一人。”

说着，斯雷因斯雷因把手上的血迹简单的擦了擦，而后转身伸向了黎骨月：“我和蕾穆丽娜明说她想杀你的想法，并不是因为提醒或是告戒，更不是试探或是关注，而是因为我想沟通——

人的语言尚且不能完美表达人的想法，如果在语言之外，人与人之间的思想与信任之间又要再添上一层隔阂，那么，人与人之间所剩下的便只有猜忌和谋算了。”

“你这样，怕不是太过坦白之后要被人反过来算计和利用的。“

我旁观着干笑道。

“也许是见人说人话，见鬼说鬼话？”斯雷因的目光变得坚定，他深深吸了一口气：“有些事情，单纯的说出来一定会被蕾穆丽娜嘲笑，所以这样的事情要做出来，比如我对黎骨月你的信任，我觉得可以从握手开始。“

因为少年笑得诚恳又坦然，故而这样与同龄女孩握手的表现，斯雷因没有一点的羞涩，反倒是一直文静的黎骨月表现得不知所措的紧张，表示自己地位有些低，不适合和主人斯雷因握手。

然而斯雷因却依然笑容温善的等待着黎骨月的回应，直到两个人的手握在了一起。

“现在因为特洛耶德家族遗产的事情，我自己说起我自己的历史大概都不会显得充分了，所以，黎骨月你也可以讲讲你的过去。”斯雷因看向我一眼，又向他身边的黎骨月示意道，

“放心，蕾穆丽娜郡主是个值得信任的人，何况，她刚才虽然说像这样的开诚布公我一定会被人利用很惨，但会利用 我的人，我相信不会有蕾穆丽娜，我也尽量不会被人陷算。”

“呵。”我没有回应，而是冷笑一声，要说利用……也许，我才是利用斯雷因最多的那个人之一吧，也许和娅赛兰都不相上下了。

我抬起左手用手背贴了贴额头，温度有些偏低，大概是因为内伤和失血。

“斯雷因，你今天的话有些多啊。”

斯雷因目光向下看了一下，而后看向我坚定道：“以前我以为，那些书上说的什么，人与人之间的默契是‘我不说，你就懂’，许多作品里都是那么吹嘘的……现在我却觉得……”

斯雷因苦笑了一下：“就像你说我和卡兰德丽都在自作多情那样，有些事情，真的只有说出来，双方才能懂。既然已经决定要把娅赛兰的想法变成现实，那么总要从现在开始，做点不什么——而不是再等别人去做，自以为是别人会懂。”

我一时有些意外，而后用精神不济的迷离眼光看向斯雷因道：“一旦她的理想落到现实，就要存在取舍和妥协，就会和她的理想产生分别——那时，你就会从她最大的理解者变成她眼中最大的对立者……

以她之名，行她之敌。”

斯雷因听完也无奈的淡笑：“确实……会吧，但我会尽量的去沟通——你说过我妄想负尽千重罪，那我总得先——炼就不死心。”

我抬手指了一下前面的方向道：“大体向前走，我们还要想办法出登陆城，现在……登陆城已经像魔方了，我睡会……”

黎骨月刚刚过来要背我，却又被我挥开。即使我知道这样在接下来的日子里会对黎骨月产生隔阂。

我强撑着马上要闭的眼睛指着斯雷因道：“还是你背我。”

之后我对黎骨月道：“利用好你的身份……”
