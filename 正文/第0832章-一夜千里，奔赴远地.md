# 第六二二章　一夜千里，奔赴远地

【蕾穆丽娜】

地球联邦标准时间，2014年5月31日夜，20点27分。

薇瑟军驻印尼某军(谐)区。

随着我的一动，所有隐藏、潜伏围绕着我原本安置的人员都因此行动了起来。我手中只是拿着一件电子板，上面有着关押斯雷因的整个登陆城近乎全貌的结构图，但并没有人员部署图。

恩米里奥自然不可能将登陆城的详细信息直接地送到我身上，然而他有他的方式，我自然也有我的手段——即使是不通过aldnoal权限的贯穿。

赤道地区夜晚的风有些大，海风顺着赤道的洋流而动，吹起我的头发。因而并不需要再像过去那样坐在机甲上执行任务，我也没有把头发扎起来，而是任由它们随风飘舞。

适时地抬手压了一下帽子，防止因为头发的飞舞把军帽也带飞。当然这样的穿扮肯定是不符合正规的军装规定的，然而贵族有这个特权，比我把军装穿的还要张扬或靓丽的贵族女子更是不在少数。

夜晚的军用机场虽然人不多，但也绝对不是安静，往往来来的有不少的战机、运输机和机甲或起或落。跟在身后的希依则拖着巨大的旅行箱，差不多有三分之二她的身高大小，然而对于有着骑士境界的少女而言，也不算是什么难事。

“谢谢。”

我将手中的电子板递给了经过我的一名军官，而后从他的手里拿过了一件看似毫无异样的电子板。

在这个瞬间，通过我手中的电子板与身上临时变更的权限，接下来将要离开印尼地区的，便不再是赛里朵亚郡主，将会是另一个伪造的身价。

而在登记上明面上赛里朵亚郡主则会转向南极的方向，再走地天轨道前往空天诸登陆城的方向进行修养。

虽然没有潜伏到核心的地方，但像这样把情报的人员安插到印尼的泛军队之中，并不是什么难事。相比于伯父等老一辈而言，薛克还是嫩了些。或者说，因为印尼本身就是有许多的贵族军队混杂在一起，故而薛克也只可能会掌握住忠诚于他的那一批嫡系部队，而对其它的部队只能起到一定的命令效果。

看着小型的专机已经在远处的机场上降落，我感慨道：“到了不得不走的时候呢，接下来……虽然头脑里有些大概的方向，但也不是完全的确定的。”

“倒是为难了希依呢。”爱乌跟着点点头道，“不过不管郡主到哪里，我都会跟随的，嘻嘻，郡主的生活上可不能没有了我。”

说着的时候，爱乌像是小女孩一样一点一点地轻轻踢着身后拉着的巨大旅行箱。

“郡主今夜一走，许多事情都会在接下来的时间里，逐渐地暴(谐)露出来了。而且我们这样，其实算是不辞而别呢，毕竟薛克世子给的准许说是明天早上为我们送别呢。”

“那估计就走不了呢，薛克也不是傻子。”我弯起嘴角笑了笑，“他说在来我之前去看过里斯芬，那一定是谎话。因为他一定会把里斯芬安排在最后，因为他一定会和里斯芬谈许多时间。”

微笑着点头向机场检查的女军官，而后带着身后拖着巨大旅行箱的爱乌走向不远处的小型专机，我一时有些无奈地淡笑了出来：“有些事情大概真的是不知觉吧，也不知道是不是我的错觉——其实在印尼这些日子，薛克找我的谈话的次数说实话……有些多。”

“嗯……我也有这种感觉。”爱乌点头应承道。

“也因此，即使称不上感情，但他也应该是对赛里朵亚郡主这个人有好感了。当然，人的天性是如此，接触得到的，总会刻意的忽视，也许是灯下黑吧，我们一走，他就一定会反应过来——我被俘虏那次他还没反应过来，但总不能再这样靠着侥幸了。”

想着印尼这一路的经历，我闭着眼睛走上专机，因为aldnoal的存在，我便是没有用肉眼看，也不会走错：

“何况，这一路来，想要从基层上，想要看到的，想要知道的一些东西，也都知道了，也不能算是什么都没有收获——地球联邦的科学家们也是很有意思呢……”

跟上来的爱乌把手中的旅行箱甩给机组人员，而后跟上我身边，在我入座之后坐在我身边，为我倒了杯温水。

“是啊，恩米里奥被老皇帝逼着要人，我们就要去搅一下混水了。”

“既然是搅混水，那么也不能完全没有收获，空手去空手回呢。”我用鼻子笑了一下，“他在纸条里没说卡兰德丽，难道我们就不知道卡兰德丽已经怀孕了么？”

专机已经起飞，而后随着aldnoal引擎的运转，薇瑟帝国的专机以极快的速度开始在夜下加速。

其实时间走的有些急，因为这一路留给我们的时间并不会很多。

恩米里奥把老皇帝要人的消息透出来，肯定不会是接到时就透出来，以恩米里奥的性格，这个消息也不可能只给我们，如果有可能，那也只可能是“第一个”给我们。

我用左手的食指轻轻敲着扶手，而后思考着事情道：“卡兰德丽在恩米里奥看来，也许会成为牵制娅赛兰手中斯雷因的好方法。斯雷因虽然现在没有什么大的能量，但潜力绝对不弱。也是如此，当初的库鲁特欧伯爵才会用那么过激的手段想办法要废了斯雷因的境界——当然，那是目的之一。”

我笑着耸耸肩：“每个人都是有私心的，就像算是满口大义的库鲁特欧伯爵也不例外。”

“郡主呢？”爱乌转头看向我，眨着眼睛里带着笑意，而后把一碟坚果推给我。

“当然也有啊，只不过我没想过那那么过分的手段。买卖不成仁义在嘛。”

“噗，”爱乌一下子笑了出来，“我可没觉得郡主有和斯雷因做过买卖呢，每次都是给那个白毛那么大的便宜。国内都有过传言说郡主和娅赛兰公主抢斯雷因一个平民呢，还好这几年弱了不少。”

我一时也无声的笑了出来，而后道：“因为有时，买卖的交易对象，并不是斯雷因，而是娅赛兰。因为，我只是希望事情——不要‘偏轨’就好。——比如这次，在我看来，卡兰德丽的存在，就可能会直接把斯雷因废掉，或者说，斯雷因会自己把自己废掉。”

“对了，郡主，明天早上，如果薛克他真的反应过来，赛里朵亚郡主就是蕾穆丽娜郡主，你说薛克会不会追呢？毕竟虽然表面看来薛克是压住了印尼，但实际上那是因为有薛克压着，可不表示薛克不在了他们不会来个二次反叛。”

“噗。”我笑着差点把喝到一半的水吐出来，而后看着坐在身边的爱乌，把头偏过去蹭了蹭少女的额角，“爱乌啊，有些市政文件建设决策那里，你代批阅的倒是还不错，和妹妹维娜一样可圈可点。其实呢，政治和这个差不多，只不过政治更多的时候，是一场博弈——虚虚实实呢。”

爱乌皱了下眉，而后有些微愁道：“那要是薛克真的追过来怎么办？”

“放心吧，他就是追来了，其实你也不用愁。至于为什么——现在多想想，或者到时候你就知道了。”我眨了眨眼睛笑道。

“呜，郡主！”爱乌刚刚举起的双手想来挠我，却又想到因为明天早上或者说凌晨的时间，只能不甘地放下爪子，“唉，郡主睡吧，我到时候和艾文或者格列维尔斯问问。”

“嗯。”我由爱乌服侍着躺在拉平的椅子上，闭上了眼睛。
