# 第五一八、五一九章　被剪断线的木偶，会变成人么？

【蕾穆丽娜】

“我……”

在少年将要把答案说出口这一刻，我的心理已经走过了许多思绪，如果说在斯雷因不知晓伊奈帆的内质与思路的大体方向之前，我可以确定斯雷因会去救伊奈帆，这很符合斯雷因的行为与思想。

那么在现在这个时候，已经确定了伊奈帆与斯雷因完全是两种看似相似却又是完全不同的人的时候，斯雷因又是否还会继续去拯救伊奈帆，就是我也无法肯定的事情了。

毕竟在这之前伊奈帆对于娅赛兰而言，在斯雷因眼中也仅仅只是有威胁，却还不是真正的有威胁。但是现在却又是完全不同，因为伊奈帆究竟会借用娅赛兰之手去做什么，便是连我也无法确定的事情——

前生记忆里动画伊奈帆并没有借娅赛兰之手做成什么事情，今生的伊奈帆显然是比前生动画里主角少年更为复杂也真实的人物。在没有可以充足参考的对象与材料情况下，便是我也不可能做准确判断。

更何况在前生记忆里，到到了明年这个时候，斯雷因确实是与伊奈帆站在了立场相左的位置，也确定是互相有着要致对方于死地的理由——并非是因为感情，而是纯粹的政治立场与利益与立场。

今生这个时候，伊奈帆已经与斯雷因有所接触——我大概可以知道伊奈帆会更可能去维持现有的体制，而斯雷因却意图在明年改变这个体制，两个人道路可能都是完全不同。

到那个时候，夹在两个人之间的矛盾便绝对不会再单纯的只有两名血脉者的aldnoal公主，而是真正的两星大势与无数立场、位置、政治、经济的矛盾纠葛。

到那个时候，相互矛盾的便绝对不是单纯的少年与少年之间的思想与感情矛盾，而是将会有围绕着两个少年的洪流推着两个人走向相左的位置，去互相斗争。

从短视的感情立场上，我确实是倾向于伊奈帆继续活下去，那毕竟是前生的记忆里的人物，他的存在也可以作为我拿来今生与前世比对的参照基准之一。

但另一方面，伊奈帆也确实是我所能够知道的不可控的人物——他绝对不是常规记忆里那些前生记忆日漫常见只有能力而没有头脑的少年男主角。

恰恰相反，他没有多少力量，相比于斯雷因，他没有变态的体质，相比于出身，他没有贵族或世家的显贵血统，相比于形势的力量，他到现在还要靠着莱艾来作为手中唯一的尖刀。

但他却又是与前生里太多acg产业男主完全不同的存在——他根本就不会按套路出牌，也根本不在乎那些我前生可以拿来作为参考的因素，更没有所谓“亚撒西、温柔”。

换个立场角度，如果现在坐在我面前的是伊奈帆，那么他一定会表情平静地说出“会借手杀斯雷因，如果蕾穆丽娜同意的话”这样的话。

很大程度上伊奈帆就是这样的人，也正是因为他有着这样的决意与执行能力，更有这样的心性与胆魄，他才能够在没有可靠靠山的情况下，一路前行到现在，几度生死而还。

所以我很期待面前斯雷因最后给出的答案——那将可能会是出乎我意料之外的答案，也可能会是我难得地深入地了解斯雷因，也是去理解娅赛兰对于斯雷因的影响的关键点——

虽然说最后斯雷因所能够给出的答案只可能是杀、不杀或者不知道这三者，但这背后着因由，却可能是千变化的。

而这个因由,才是关键。

“我……还是会救他。”斯雷因停顿了很长的时间，前前后后张开口又闭上了，足有三十多秒之后，少年才把答案说了出来……我虽然不可能借由aldnoal场以知晓别人心中想的是什么，但想来，少年得出这个答案时心理里也满是犹豫吧……

但是比我的思绪更快的是面前斯雷因从得出这个答案之时的犹豫，之后便在表情与精神上迅速地走向了更为坚定的立场——也许少年要做一个决定的时候可能总是犹豫，但是少年一旦做出了决定了，便表现出了不会再犹豫而浪费时间与机遇的坚决。

斯雷因如是，伊奈帆也是如是，能够在下定了决意之后便不会再有迷茫地坚定执行下去，而不是如我一般的普通人在行动的过程中依然有些患得患失，确实是少年人可以有所成就的特质——也对，如果斯雷因和伊奈帆没有这样的特质，那么早就不知道死在哪次的战场或意外里了。

显然斯雷因也知道单纯答案我不可能满意，他需要向我解释他心中的答案，以回应我之前为斯雷因做出的关于伊奈帆的分析与解答：

“我想，杀人，只是保护她和你的手段，但绝对不是毁灭他的目的。

如果我在这个过程中迷失了意义与根基，那么我最后也只不过会是沦为一个为了目的而目的的行尸走肉罢了。“

少年的声音平静中带着坚定，虽然声音不是很大，但确实是显现出少年在经过了许久的心理迷茫之后的坚定与清明。

“纵使我之前的救 的可能不一个‘人的内质’，纵使我以后可能还会有这样的情况要面对，纵使我可能真的要回到过去，再去面对一次同样的情况 ，那么我纵使在知道了之后，我也依然会去选择给她打电话，让她来救他。

因为在这种情况下，我只是做了一个微薄的工作，所拯救的便是一个‘文明意识的内质’。

我们人类是文明的一部分，但也确实不是宇宙文明的全部。也许伊奈帆可能无法在人类的文明体系里成为文明的一员，但却一定可以在我们所未知的宇宙文明里成为一员。

也许可能他那样将所有文明生命都当作‘存在’来对待确实是有其冷酷的一面，但是换个角度,那样的冷酷也确实同样是对待着他自己。

所以他依然是个值得尊敬的人——他纵使与一般人可能有所不同，但却依然是一个对待包括他自己在内的所有人，所有存在，从意识形态上平等的人，而不是一个表里不一的人。

那么我自然应该尽我所能，去在同样的情况下，再做一次同样的选择。“

斯雷因越说越显得心理放松而从容，也越说越显得目光清澈而透明，在这样的言语中，我确实是看到了少年在谈吐之中在渐渐地摆脱了过去目光都围绕着娅赛兰的局限而渺小，而是在有了过去几个月的经历与理想、认识、知识、目标的支持之下，渐渐地变得更加完善的一个人。

一个可以让我视之为人的平等的存在。

“虽然我和伊奈帆如果从内质来看，可能不能算是完全的正常人，但确实也是在这样的社会环境里存在着。人的本质有时并不一定完全依赖于他的血统，还看他所处的存在环境以及他在这样的社会环境所做出的作为。

也是因此，可能在某些人看来伊奈帆的思想确实如你所解读那样，非我族类其心必异，但是，这也只是站在人类的角度上来看待的。

站在一个更为长远的角度上来，又有谁能够说得清楚，所有的事情的选择一定都是正确的或是确定的呢？

我们所能够做出的选择，既因为我所能够看到的目光而有限，所做出的选择自然就是短期的，片面的，有限的，只不过是在尽可能长远的选择里,我们在尽自己的努力。

但是我们依然不会因为这样的选择就只是会留下后悔——一如蕾薇你对我的帮助，也不是从一开始就确定能有所收获或是收获……但是你依然这样做了，所以在这样的情况下，我也同样也应该要向对你说声谢谢。“

“噗”我笑了出来，“照你这么说，那看这个样子，伊奈帆下次有机会再见到你还应该要对你说声谢谢呢。放心，我会让人转告他的。”

“嘛，那个就不是太用了，也就是一点举手之劳，而且真正帮助到他的是你，并不是我。”斯雷因的表情微微的腼腆，而后道，“不得不感慨许多先贤之人的学说真的是很厉害，我也只是在他们的肩膀上有所学习，才能够有今天的这样的认识，而不在再像过去那样只是看到的有限的知识,而后只能做出有限的判断……无论是无产阶级科学革命学派创立者的辩证法，或是华汉开国之人的矛盾论，于我而言都是受益菲浅。“

斯雷因说到这里也只是自嘲了一下：“当然，这也说我们作为个体的时候的无奈和悲哀，我们总是在纷杂世界里做出于我们自己所见的可能限制里的有利的选择。

只不过是这个选择有人所见的尽可能短期的眼前，而有人所做的是尽可能长远的未来。

蕾薇你会在这样的情况下帮助我，不得不说，不管如何，你做出了一个在你看来更长远的选择，纵使这个长远的未来连我自己都不自信，但你依然相信。

不是先相信我自己，而是去相信着相信我的娅赛兰和你，而后在从中明白去相信我自己。

是有些绕口令的奇妙呢。“

说到这里的斯雷因看向了窗外，而我也把目光转向了窗外。在那里，娅赛兰的专车已经抵达了医院，相信不久之后，伊奈帆的手术就会展开，那位总是在钢丝上跳死亡之舞的少年，又会来一次大难不死吧。

斯雷因叹了口气，说出了我之前没有在意也未曾想过的,或者说我可能在潜意识注意到却少有放到表意识上来的内容：“我想，你愿意帮助括括我在内的他人，便是因为在你看来，虽然可能是非人，但依然是人，或者说依然有所意识，有所思想，有所决意，同样也有所行为的存在。

以前我不太明白，我以为你对她的态度会是很激烈；也是因为很激烈，所以你不愿意与她见面。

但现在想来，其实你纵使要面对她，你大概也只是会表现得很平静，你不愿意见她，只不过是因为你会觉得麻烦，而不是会因此而感到敌对或愤怒……

立场虽然有所相左，但不表示一定是充满情绪化的愤怒或冲动，更不表示因此要充满傲慢与偏见，这样稳定自己看待世界的目光与视野，才能够有所成长……

这是你和伊奈帆可以做到的事情，也是我和娅赛兰现在大概无法做到、做好的事情，是我需要学习的。“

这些日子，面前的少年与我之间有许多的交流，而随着交流的深入，少年也从原本的寡言少语变得能言健谈了许多。这些健谈也并非是什么无用之言，而确实是流出自少年心中的想法，因为信任，所以把这样的看法与观点言语向我解释。

“嗯。”

与之前的我会做出一些回答有所不同，现在的我也只是应了一声，于是餐桌也从之前的我和斯雷因的对话之回到了安静的沉默。

只不过这种沉默却又是与无言不同，斯雷因大概还需要一些时间来整理他自己的思路，而我同样也要梳理一下今天他所说出的这些言语。

到这个时候，那位在病床上的伊奈帆反倒显影不是那么重要，因为他会康复出院已经是板上钉钉的事情，只是不知道在伊奈帆这样的一次谋划里，又是否要与我的接触，而我也不打算等他出院或是接受他出院之后的感谢。

……可以说，斯雷因会选择救伊奈帆，有很大的原因，是因为斯雷因看待这样的问题，不看出身构成，而只看行迹所为，和伊奈帆会先做出有所划分，再做出判断是有所不同的。

从这样的言语观点里，也可以看出思路的……他会是与我前生记忆里的斯雷因是不同的人呢……前生记忆里后来登上了帝国贵族舞台的伯爵少年是个很激进的人，也不惮于主动发起对贵族或是对联邦军队的进攻。

可是现在的斯雷因呢？

斯雷因他明显不会去做第一个出手的人，而是会一定选择去做那个“先吃亏的人”。他不会开第一枪，但少年是否有实力与能力能够践行“不让对方开第二枪”，这样决意也要打个问号。毕竟信心实力与实际的行动之间往往是两回事，并非是完全等价。

但要说这样的一次行程真的毫无意义那也并非如此，虽然没有收获什么实质的物质与资源或科研的财富，但是于我而言确实是在围绕着斯雷因的行动之间做出了重大的一步——

意图让斯雷因从围绕着娅赛兰转变成围绕着蕾穆丽娜转是不可能的，那只是从一个没有灵魂的傀儡变成另一个没有灵魂的傀儡，更对不起我心中的良知与前生的记忆。

斯雷因他在这样的地球之中渐渐地变得开朗、自然，也能够把许多原本压抑在心底的事情都说出来。这是好事，因为他是那种会把所有的心理压力压在心底而后自己又不能能从意识表面感到心理压力的人——但是人的心理压力却又是必然存在的。

否则也不会有当初假娅赛兰遇害时，斯雷因那样发狂。

少年在这一路的成长阅历，能够一点点地累积思想，深化思维，也能够逐渐地建立和打磨他自己的坚定意志。

至于他心中是把如何看待他自己的，又是把自己放在一个什么位置这样的哲学之问“我是谁”，虽然也可以现在提出来，但确实还是拔苗助长。

相比之下，让斯雷因自己在这尚且还有的时间里一点点成长，相信他也会在这样的成长中，逐渐地以他自己的意志，向他自己的内心提问，然后由他自己来解答吧。

……虽然有些不那么甘心，但却又是客观的现实，娅赛兰对斯雷因的心理影响能力和开解能力，确实是要在天然要胜过我许多。

但这至今而来的现实，也确实是让我所释然。

因为我虽然没有把斯雷因从娅赛兰手中的木偶变成我的木偶，我也不希望这样，而我确实在这样的一路之行中，把娅赛兰手中连着木偶斯雷因的一根根线剪断。

也许这确实让这个一直在她手中起舞的少年在失去了引导与无形掌控力量的少年因为“无力”而摔倒在了地上。

但他确实又在摔倒了之后，没有了太多根连着娅赛兰的木偶线。名为木偶的少年斯雷因，再次以自己的意志行动起来，跳下了木偶戏的舞台，然后推开了木偶寻址连接外面世界的真正的大门……

去迎接现实的光，而不是木偶舞台剧里的虚假的灯光。

他会选择从木偶变成人，然后再回到那里，去面对娅赛兰。

这也是为什么面前的少年可以一路平静地跟随我而行动，而不是每时每刻都急躁地想要跑到娅赛兰的身边像归巢的鸟。

少年不是鸟，而是人，他确实需要有他自己的想法……成为一个独立自主而全面的人……

桌子对面的少年放了餐具，虽然远眺的目光有着一些追思与向往，但却没有渴望与失控的迷恋，而是更多的平静与从容，淡定与镇静。

面前的斯雷因收回了目光，看向我道：“我想，我们下午就会离开这里吧？”

“这么急么？”我微笑道，虽然这样问，但我心中已经微微地预料到了答案。

“嗯，也不算是。”少年点点头，平静道，“不仅我不适合在这里见她，我想你也不想见她吧。”

少年又笑了笑：“当然我想，也有你不想见伊奈帆的原因吧。因为怕见到他对方又说什么而你又不得不赞同。“

“嗯。”我并没有否认，毕竟伊奈帆有这个能力。

“不得不说，虽然他现在的位置依然很低，但是很厉害。”斯雷因感叹到，而赞许对手从来不是丢人的事情。

我起身，爱乌则为我披上外套。

“承认对手厉害并不是丢人的事情。”我点点头。

“那么薛克呢？”我笑了出来，毕竟似乎薛克一直把斯雷因当假想敌，但斯雷因却并没有那么想过薛克……一如斯雷因一直把伊奈帆当假想敌认真对待，可是伊奈帆对斯雷因却是战术上重视战略上蔑视的……真是微妙的链条，我不由得在心里暗笑了出来。

“应该也很厉害，当然，我对薛克了解的不多，也无从做出太多的判断。”斯雷因披上外套耸耸肩道，“毕竟我的成长说实话，还真的很有限。”

“是么？在我看来倒是不少了。”我笑了出来。

“啊……谢谢。”
