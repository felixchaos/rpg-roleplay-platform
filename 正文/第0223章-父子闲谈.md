# 第十五章　父子闲谈

：自己玩去吧，是哪个混蛋干点击不投票的产生了我是义务按时更新的规则？逗我呢喵哇！

【无】

2014年3月12日，夜。

“皇帝陛下御令：〈巡游军团于火地多条轨道上牵制封锁地球联邦的星野舰队和卫地舰队。〉”

深蓝长发与黑曜石般深邃瞳孔的男人一身华贵黑衣，站在高楼之上，仰望着同样深蓝的天空中无尽的群星，而在他的身边，则是一张巨大的全息投影，果决刚毅的年轻人站在全息通信的另一边，看着手中的战备材料中军事部署与安排，对于年长者的问话并没有作出回应。

“既然是进攻方，理应在战局上先手取得优势，如果一切都以地球联邦的考量来做预备性的攻击，那么所谓的先手就会渐渐地失去意义，即便轨道骑士看似先发制人。就像两名顶尖高手对决，有时决定胜负的并非双方实力，而是谁能够抢占先机，以快打快，以快压快。只要能够一步一步地压住对手的节奏与速度，就可以逐渐地积累优势……和地球联邦曾经历史上的闪电战差不多。”

英俊冷漠如黑夜王者般的男人如此多话，也换不来好像一个模子里刻出来的少年人一点回应。

“你和蕾穆丽娜的婚事，我准了。”年长者，亨舍利－乌里克欧兹一反之前对两星之间局势的看法，转而谈起了儿子的儿女私情。

薛克微微地挑了一下眉毛，向身边的人下属指示一下作战安排，而后沉声道：“我本来以为你会一直反对，谁让你当年被蕾穆丽娜的婶婶耍了一遍，说是要和你结婚结果却先于你死了，于是你自然也见不得儿子好。还有——我喜欢谁，我追求谁，皆是由我自定。”

“呵，你都知道了？我看着很像？”亨舍利用鼻子轻轻地笑了笑。。

“啊……很像，至于你当年的糗事……自然是花了不少的时间都打听出来了，也不枉我当初自己离家出走。”画面一阵晃动，显然是薛克拿起了电子板，要去什么地方。

“我当初先是反对你和蕾穆丽娜婚姻，再是因为你杀了自己的几个庶出的兄弟姐妹，打得逼你离家出走，都是等着现在。”亨舍利用扶手拍了拍栏杆，凭栏眺望赛亚尔的方向，而后无奈地摇摇头，“你倒好，杀得一点都不手软，就不怕我一怒之下，顺道借着家里其它几个叔伯的压力连你都杀了？”

“哧，别说笑了，你忘不了蕾穆丽娜的婶婶，也忘不了我娘——只要你透过我的脸能看到她们两个人。”薛克来到了一间小型的会议室，而此刻早已经有几十名下层的军官贵族或骑士就座，有老有少，但无论哪个男女，看着都比薛克年长，却皆正坐谨慎。

薛克招了一下手示意会议的书记主持会议，就接着保持和自己父亲的通话，很显然在坐的都没有这位侯爵和世子的身份大，也就没有谁会真的有资格对这对父子的谈话指手画脚。

亨舍利沉默了数息，很显然也在回忆着什么，而后道：“雷利加利亚自恃aldnoah的发展，以为有着十四年的积累，再加上当年的heaven’s_fall的失败在他眼里只是个意外，便认为地球是囊中之物势在必得。”

“如果巡游军团不迅速插手，两星的战争就可能会打成僵局，而一旦战势拖久了很显然对我们不利？”薛克微抬了下眉毛，这句看似疑问 的话语很显然带着肯定的味道。

“没错，一旦战事胶着，即使我们本星的后勤跟得上，战争对人员的需求也跟不上。”

“死不起？”

“没错。”亨舍利回忆一下，而后道，“你应该对扎兹巴鲁姆的战法非常得有印象，他的风格就是〈快〉与〈致命〉，任何拖沓的战局行为在他那里都是严禁的。”

“所以一旦我们这边被地球联邦拖着以命换命，后期即使装备的生产没有问题，所需要的后勤、士兵、驾驶员、指挥员也不够？”

薛克右肘支着桌子，用左手捏着下巴陷入了深思：“很显然我们这边的兵员是死不起的，如果按照其它轨道骑士那样自恃aldnoah就可以大摇大摆单骑或少骑行动就是非常不智的行为——”

少年人向站在讲台前的女子书记军官抬起手道：“以至少七人为底线的小队集团行动，严禁所有的个人英雄主(谐)义行为，一旦发现多于己方三倍数量的敌人都要向指挥中心、附近空舰编队或者登陆城进行汇报并请求支援。”

“但是其它的轨道骑士可不会听你的。”

亨舍利笑了一下，看着自己的儿子渐渐地成熟稳重起来，虽然有时依然喜欢兵行险招、行事霸道，但很显然在地球联邦这块险恶的土地上兵行险招，肯定不是自与薇瑟帝国交战起就一直在兵行险招的地球联邦军(谐)队的对手。

“凯特拉特塞已经前往华汉洲区了，甚至直接进攻的首都北京。”薛克皱起了眉头，用手揉了揉眉心，虽然薛克凭着在军(谐)队中的努力，家族背后的支持和本身的身份已经达到了一定的高度，但想要像那些位高权重的轨道骑士在战争中直接地左右军(谐)队分配，他还缺少些大的军功和需要的名望。

——毕竟他现在是在轨道骑士这个军事团体中白手起家。

（（轨道骑士……对老皇帝还真是忠心耿耿啊……））

一瞬之间就选择了与实权贵族看法相反的北半球，而且还是直奔华汉洲区的首都而去，可见其……勇气。

“华汉人会撤去沿海，然后等着对库鲁特欧来个关门打狗。”亨舍利转身离开了高楼的看台，身后的娇美侍女则将悬空投影用的电子板抱在胸口，丰满的胸部在透明的电子板上挤出圆润yòu惑的形状。

“会是一场苦战，但……库鲁特欧虽然有意表现忠心与勇武，却实则……胜算渺茫。”亨舍利就是当着薛克对面会议室中所有的中下军官青年贵族和骑士预言了 一个令人不快的结局。

“守不住？”

亨舍利由两名年轻甜美的侍女服侍着换上了出门的外套，薛克虽然对自己的父亲半夜出行有些在意，却不会真的说什么——不管他是去半夜抽查下属机构，还是去高级会所见哪个情(谐)人。

“如果你以为自下而上，自南半球向北半球穿插很容易，那就想得太简单了——首先从最新收到的情报，星野舰队和卫地舰队前梯次舰队在公主出访之前就一直在封锁着北半球的天区和整个太阳系的火地轨道之间的大片上半天区——孤军深入北半球的部分轨道骑士不会得到直接快速的支援……虽然也不会马上断炊就是了。”

坐在车上的亨舍利将双手伸进两名侍女的衣领之中，而那两名丰润侍女则抱住胸口，用女子丰满的胸部为亨舍利暖手，从薛克与父亲通信起，父亲身边的侍女容颜就没有重复过。

画面固然香艳，但薛克并不为所动，毕竟在遇到蕾穆丽娜之前不懂事的年纪也曾经如此荒yín过。

“入侵南太平洋之后，马上构建联合体系，务必将南太平洋、南美洲区、澳洲洲区、非洲洲区联合成军事快速传递网络，至于向北的扩展，则尽量收敛。”薛克起身，指向那名女书记军官，“你继续主持会议，按照我之前写的计划，我出去一趟。”

亨舍利坐在专车中，看着风风火火的儿子在登陆城的通道中向着指挥中心的位置奔去，笑道：“干什么去？”

“如果按照你说的，那么我们会遭到的反抗根本就不他们所预想的那么简单，而是地球联邦对于南太平洋的压穿之势，虽然我们同样也反过来去穿插对手的华汉洲区，但我早有耳闻华汉洲区在各个山脉和地下都有华汉陆军的导(谐)弹基地，而显然我们不具有主场优势。”

“想让他们把登陆城和空舰的联合进攻阵形改成防守阵形？”亨舍利看着儿子轻笑一下，似乎回忆起了那个风风火火的扎兹巴鲁姆家的郡主，以及那个温柔婉约的妻子，“你说服不了他们。”

“理由。我身为侯爵世子，又是一直以来凭借着自己的能力提高到如今的军衔位置，就算没有直接地完全干涉权，但也应该有几分参考的价值。”

“呵，轨道骑士以为建立南半球的军事联合网就能占领一地，他们就忘了背后的南极洲？”

“图拉克亚监察伯爵和立曼顿总领伯爵会直接占领南极洲。”薛克停下了脚步，向父亲反驳道。

尽管他对于那个一直对自己不管不问，以放养方式处理自己的父亲突然关心起自己很是不满，但不管怎么说，父亲也是和扎兹巴鲁姆交手数十年，就算没从扎兹巴鲁姆那里讨过便宜，但经验绝对不少。

“嗯，这个可以舍去，我们说点门外话，你对于华汉洲区各洲区出身的军(谐)区素质怎么看？”

“北美洲区的军人综合实力出色，欧洲洲区的军(谐)区技术不错，华汉洲区的军人纪律严明，俄北洲区的军人悍勇十足。”

“反过来，各洲区的军人也可以说都有问题——北美洲区军人的同性恋问题一直很出名，澳洲洲区的军人就是纪律问题，华汉洲区的军人在平均技术水平上有所欠缺，既可以说他们是服从纪律，也可以说他们是被消耗快的——消耗品，至于俄北洲区的军人……不仅历史上有屠城的恶习，再者易冲动，别不相信这个判断，人在战场上杀红眼什么都能干得出来，当年扎兹巴鲁姆就这么玩过对手。”

亨舍利坐在专车中看着夜色，由两名甜美侍女主动为他喂食：“所以轨道骑士也不例外——雷利加利亚想着让轨道骑士快速推进，拿下地球联邦，也不想想，轨道骑士在军事素养上真的能和实权贵族的军(谐)队相提并论？”

“巡游军团一直通过内争来反复自生死之间锤炼军力，同样的星野舰队也经常在太阳系内轨道反复军事演习，本海舰队和天梯舰队作为被侵略方自然会有初期的士气支持，甚至随着家园被一步步侵略，士气还会积累，而入侵的轨道骑士军(谐)队……就算有着公主 的被害作为起点的意志支持，也会在进攻的过程中被腐化或遗忘……”自言自语的薛克很清楚地球，那个蓝色星球的美好。
