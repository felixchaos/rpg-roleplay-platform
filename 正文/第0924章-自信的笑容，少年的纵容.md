# 第七十九章　自信的笑容，少年的纵容

【蕾穆丽娜】

我伸出食指把桌子上水点画出圆圈三角或方形螺旋：“斯雷因你觉得她进境真的比我差么？并非如此，这次旅行前最后一次我知道娅赛兰境界状态是三权初境转中境，而我的状态是三极初境转中境——如果按照资质来论成长速度，其实她接下来境界会走得比我还快。”

“这样啊……”斯雷因感慨道，“但她自己依然缺少自保能力吧？”

“嗯，权共鸣本身就是增强aldnoal在体系扩展和相应aldnoal设备发挥以及权力、效率，这方面她有如增幅核心，而极共鸣却做不到更好。当然，权共鸣因为身体不能将能场增幅外放，脱离了设备仪器确实没有太多对外的直接干涉能力。”

“哼，要不是我们小姐曾经从三极巅峰跌境过一次，又怎么会落得和那个黄毛公主一个境界。”爱乌为我打抱不平的翻了个白眼。

“好啦，那次境界提升本来就不稳，如果走差了可能就是灰飞烟灭呢。”我宽慰身边爱乌道。

“还是小姐好，来，啊～”

显然斯雷因是第一次听说跌境，他惊讶而神情微有担忧道：“跌境？还有这种可能？”

我笑着耸耸肩，边吃爱乌喂饭边歪头笑道：“跌境很意外么？或者说啊斯雷因——你什么时候产生了aldnoal一定会是一帆风顺的错觉了么？跌境、虚境、爆境都在帝国历史上有过，不然你以为众皇子公主都是正常死亡或死在战场上么？

会一帆风顺，那是因为娅赛兰她——我那个同父异母姐姐资质是全帝国最优的，你应该知道她是兄妹基尔泽利亚皇帝和图莎依奈亚公主的孩子，而她们俩原本是皇子和公主。”

“近亲……很近的？”斯雷因的脸色有些尴尬，“我以为本是远亲或妻随夫姓……“

“是近亲啊，而且是同父所生近得不能再近的血亲了——纯化血脉才有你后来知道的娅赛兰。”

显然过去看过许多书的斯雷因就是因为身处薇瑟帝国，反而错过了帝国自身某些秘史，更何况一些帝国的内史也不会在教科书上讲，而是贵族们的藏书。

毕竟图莎依奈亚的身份，在她成为皇宫前，作为公主是只有贵族知道，并不向平民公开的。

这么想，荷尔佳这种和男人竞争的公主，也是帝国上少数的几个之一呢。

直到我们这一代，因为没男人，老皇帝祖祖父就算想把她隐藏起来也没有皇子站前台。

我突然恶趣味，右手支着脸颊看着面前斯雷因道：“那一千亿斯雷因你需要的时候随时可以和我说。那么我换个问题，如果娅赛兰她跟你借那一千亿呢？”

我微笑得弯起嘴角：“就我所知慈善是很烧钱的，尤其是像她那样。如果她和你要钱呢？如果不是借而是要呢？”

当我自己问出来之后，我自己都愣了一下，而我也看到了坐在我对面的斯雷因同时陷入思考。

低着头的少年似乎是在抉择，他没有注意到我的表情，而我自己却从最开始自我惊讶变成内敛低眉……

因为这种问题正常来讲我本是不应该问出来的。

明明知道这种事情其实在未来是一定会发生的。

我亦有我的给自己划定的提问方式……我会批评斯雷因，会反对娅赛兰，甚至会嘲笑、冷讽他和她，但绝对不会给出这种诛心的问题——

因为这还有一个隐含的前提——

娅赛兰会不会和斯雷因要这笔钱？

这个提问已经假定了娅赛兰会要了。

如果不是借，而是要呢？

对娅赛兰而言，我那个从小含着金汤勺出生的姐姐可能连金钱与贫富的概念都没有体会——她总是说要帮助困难中的人，但我从来没从她身上体会到那种“被限制、被约束、被困顿、被无力”时带来的贫穷无力感。

她的身份决定了她有这样的资格。

但在安逸的皇宫之外，世界却又并非如此。

所有的人都在与天斗与地斗与人斗与万物斗。

这个提问诛心在于——向斯雷因内心捅刀的人不再是我。

如果要嘲笑或否定斯雷因什么，我从来都是堂堂正正地来——因为当距离产生的幻影被剥离之后，怒其不争会让我有话直说。

我本就不在意斯雷因会不会喜欢我，因为我所期望的本就是他能有个善终，他把他自己坚持的道路走完，对得起他自己，也对得起未来那些追随和支持他的人。

而不是仅仅为了娅赛兰，他负了追随他的天下人，那样就好。

这个提问的诛心在于——我假设了一种情景捅刀的人是娅赛兰。

目光微低看着餐桌的少年沉默了数秒，其间他的右手大拇指左右缓慢擦动过桌面，那不是有意义的动作而是表明其人心中是在抉择与思考。

“同意。”

斯雷因又道，

“但我不会把特洛耶德家遗产的那些钱直接给她。”

我狠狠地瞪了一对方一眼，什么时候也学会大喘气的说话了？

“你自己有钱么？说同意借，你自己没钱，她真的借还不是动遗产那些钱——虽然看来可能那些令尊都没放在眼里，只是当小头。”

眼前的少年抬头再认真看我的眼神中带着坚定，然而被斯雷因这样直视，虽然我有些不习惯，但还是尽量回视对方。

与过去在火星时他直视我那样仅仅只是朋友之间的目光不同，现在他的目光中带着深切的信任：

“我自己确实没钱，所以如果她真的的需要钱，那么我自己则要努力，去赚取那些钱。超出的部分我会告诉她我暂且无能为力，她请等一等。

当然，我自己对钱没有丝毫的欲(谐)望，蕾穆丽娜你说的那一千亿就继续放在扎兹巴鲁姆体系中央银行吧。”

他缓缓吸了口气，又缓缓地将其放出，其间可以看到少年纤细却坚韧的肩膀上下微微浮动：

“那是家族长辈的钱，那是家族故人的钱，但——不是我的。

因为我还没明白家族的使命和目标，我一直和特洛耶德这个名字、特洛耶德这个家族之间是有些分离的。父亲也从来没有和我谈起，我连父亲在火灾中丧生的最后一面都没有见到，所以一切的因由和他的，他们的想法，都无从得知。

所以现在这次旅途，我想知道。“

“令尊过去平日也没有说过什么？”我伸出手示意斯雷因握住我的手，激起aldnoal透过他的身体。

少年回应我伸出的手，两人个人双在桌上相握，这要是一般人看来可能以为是情侣吧？

“父亲和我谈起的从来都只有科学、技术方面，涉及到人情、政治或是目标、战略的事情，从来都没有说过。“

既然连aldnoah激发都不能唤起斯雷因的回忆，那就是记忆中真的没有，或者是被其它条件封存。

“所以这些钱，即使我花了可能也不会用在祖辈们期望的地方。

我自己，依然需要我自己的答案。

祖辈留给我的，是赠予，不是我无所作为原地空等坐享其成的借口。

那些说是我的，也可以说不是我的。

只有自己能够用自己的能力与力量创造、争取并握在手中的，才是自己的。

所以，娅赛兰亦需要钱，那么就需要我自己去挣，然后在她需要时交给她，而不是去支用不属于我的，去践踏祖辈们的努力和积累。

就像蕾穆丽娜你说的类似的……嗯，是我的，终是我的，不是我的，终不是我的。“

斯雷因说话时带着笑意，这个笑容中带着重建自信与从容，也在渐渐扫去过去一直遮盖在他神情中的阴霾。

斯雷因这种原本我以为只会在一年后月球碎片基地中的笑容，现在却出现在这里，反而让我脸有些红……嗯，是因为这些年身体已经习惯了女性么……

不得不说斯雷因的笑容确实对女性有杀伤力，然而我仔细思辨亦明白——

眼前的笑容并不是我记忆中前生成为月球碎片基地总指挥时步入十八岁，从少年变青年的那个笑容。

那时的笑容，表面无奈温柔中内里深埋着一般人难看透的冷漠指着与绝决，带着为达目的的不择手段。

而不是现在这样，笑容中的带着暖阳的光。

这个笑容是现在这个世界的他独有的。

和娅赛兰有些不同，亦有些接近。

少年在缓慢却坚定蜕变与进境却是看得到的。

……是因为性别歧视么？我本应该为斯雷因染上娅赛兰的阳光之色而感到生气么？可这样能够自我开朗开解的斯雷因也是难得一见，只可能在人幻想的暖阳斯雷因。我不应该为此感到高兴么？他终不是记忆中的孤岛结局。

或者说……不以为喜，不以己悲？

或者说……不为之狂妄，不为之自鄙？

说不上来呢……

“呀。”

我一时叫了出来，因为一双柔荑贴到了我脸颊两边。我这才反应过来是爱乌把双手捧到了我脸颊的两侧。

“小姐你又发呆了，还有斯雷因，赶紧把你的笑容收起来，你是想对我们大小姐用美人计么！？有你好看！”

“呃，我笑了么？抱歉。”

少年摸了一下自己脸颊，表示并没有感觉到自己在笑。

“抱歉，如果我刚才笑了我道歉，毕竟和人说话突然笑起来我想应该很失礼的。”

斯雷因又重复道。

“还说没有！我们大小姐脸都红了！还热的呢！”爱乌嗔怪道。

我把爱乌的双手拉开，甩甩头，这长长的头发也跟着左右摆动，发丝因此有些微乱，爱乌立刻掏出梳子我为简单理了起来，而我则看向斯雷因……又把目光往窗外偏开……也不知道这是因为设想还是什么，我道：

“如果……如果她不想还钱呢？或者说，你刚才的意思是……不需要她还？”

斯雷因却十指交插想了仅仅一瞬，就坦然宽解道：“如果这是她所期望的话，没有关系。”

我无声叹了口气……

娅赛兰可以这样为所欲为，就是因为背后就伊奈帆和斯雷因这样的人吧。

我将双手放到自己大概还有些余热的脸颊上用力挤了挤，这样也不会显得我是因为对面的美男计脸红的还是自己揉红。

身体反应的事情在这具身体里被激素影响这些年——理性不会被干涉，但感性却会被潜移默化的影响——我也没有办法啊。

我重新与对面的少年双瞳对视，声音中没有多少感情起伏：

“既然斯雷因你已经重整好自己的状态了，那么下午时我们就边走边说说一个我曾经提过的话题——

建国时的薇瑟……或者说火星人时怎么熬过当年的饥荒和封锁的。“

我的笑容中的带着冷意……应该说是冷笑吧。

“毕竟，那时的火星可没什么凭空无中生有的粮食工厂。”

客观现实可是从来不会因为谁的感情而转移或改变。

我突然想起看过的小说中的说法：

“斯雷因，无知与弱小从来不是生存的障碍，傲慢才是。

——希望你也能记住。”

也希望我自己能记住。
