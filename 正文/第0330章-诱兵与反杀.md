# 第一二一章　诱兵与反杀

星：星（推荐票）

1050：20091131050（推荐票）

【无】

当星点越来越多之时，薛克眼中蓝天穹顶，都是一片炙白——

那是炮弹多到近乎密排一般的程度。

前方十几公里处，数百有人与无人机构成的能量回旋阵形如一片胶水死死地缠住了一百七十多架领长机甲。

（（以这样的炮火密集程度……扈从机甲和战机抵达之前，就会被地球联邦的炮火洗干净。））

围点打援，对面的前线指挥官即使是处于弱势之中，也可以借由后方空舰和海舰编队之手，构造出一场对薇瑟军队基层力量的杀局。

也许此刻这些已经被斐柯伦等人视为肉盾而舍弃的战机和扈从机甲不算什么，但他们都可能是未来的领长机甲驾驶员。

“扣下。”

薛克一声冷血的命令，原本围绕在其周围的领长机甲小队团体便直接出动，阻止了大部分的战机和扈从机甲团队的行动。

“〖薛克－乌里克欧兹世子你在干什么！〗”巴鲁克鲁斯看到后方薛克竟然截援，不由得怒吼道。

“你们太冲前了，将士断腕理当进退有据。而且你们这种调令是在无谓地浪费战力。”薛克皱了下眉头，便平静地回应道。

就如蓝发的青年所言，当如群潮的部分扈从机甲和战机编队即使以最大的速度前线突进，力求在地球联邦的「「轨道偏压弹」」坠落之前突破时，一道道粗大圆柱体的炮弹已经以直角坠落的姿态，轰砸在薇瑟帝国单位的上方。

钢铁与火焰的爆炸陡然之间在薛克的眼前炸开，一道道白色的炮弹轨道白线如暴雨般密集交织，有如曾经的青岛战役中「「喀秋莎动能阵列」」的密集轰炸再现。

大片的海水被掀起，冲击波被扩散开来，火焰的花朵与红云肆虐四方。

如果渊戮骑士孚相生在这里薛克还能挽救下这批在炮弹之雨下冲刺的帝国军队，但此刻，他也无能为力。

青年看着眼前一片战机与机甲陨落的火海之雨，目光平静，黑色的瞳孔中倒映着死亡与钢铁的尸骸，无动于衷。

而就如薛克之前所言，那些所谓的支援战机和扈从机甲，当穿过了炮火之雨后十不存一。

而这样的残余力量，立刻被地球联邦的远程空舰火力与战机所剿灭。

“〖呵，巴鲁克鲁斯卿，我等有何可惧！且看这几支地球联邦空舰编队早已摇摇欲坠！〗”对薛克之前的行为有所不满的斐柯伦伯爵则驾驶着攻杀型领长机甲指向着前方虽然战机不多却在空舰数目上残存数艘着装甲空母。

这其中，定然就有那个地球联邦的分队指挥官。

“〖贤婿，战阵之上，一鼓作气勇猛精进，这些老一辈的经验，你可得学着点。〗”与薛克的保守相反，斐柯伦早已看到了胜利，只要顶着这群如苍蝇般的战机干扰将眼前的仅存六艘空舰击溃，他们就可以将战线推前一大步！

薛克并未言语，而是冷眼地看着前线战场上斐柯伦等人的厮杀，指挥着手下的力量与之前刚刚截获的战机和扈从机甲编队分散向其它更为重要的小战场上给予支援。

薇瑟帝国缺的是人口，像斐柯伦伯爵等人之前那种不把平民骑士和驾驶员当人看的行为，薛克既不会愤怒，但也不会无谓地赞同——他们，本应发挥在更有价值的地方，即使是牺牲。

而原本还在与斐柯伦伯爵等人纠缠的战机、空舰、装甲空母团队也开始后撤，能量的消耗，火力的消耗，战机的维护与支持，诸多的力量损耗已经使得克拉斯克一方显得后劲有所不足。

看到了地球联邦前线舰队疲态的斐柯伦等人却在无比欣喜，因为在这场力量的对耗结果，确实是他们赢得了最后收割的胜利！

便是之前无比棘手的那两支沃尔洛德精英机甲小队也在百多的领长机甲追杀之下不得不左右回避，离开战场的中心，游移在外。

三位伯爵早已经看到了以最高不过熔殇骑士的直属领长机甲团队围杀沃尔洛德是何等令人欣喜的战绩。

“〖克拉斯克……你退了！？〗”李由看着后方原本作为中继火力支援与大后方空舰运动火力配合的装甲空母编队缓缓后撤，而薇瑟帝国的领长机甲团队甚至放弃部分针对战机的目标，对空舰穷追不舍，这个中年男人没有任何的“队友临阵脱逃”的愤怒，反而是……惋惜。

未有经历战场的普通人可能认为，临阵脱逃会生存率更大一些，但如李由和克拉斯克这样的经历数百次战场之人，早已经清楚，那些先跑的，才可能是敌人首先打击的目标。

“嘛……”金发蓝瞳的高大中年男人叹了口气，心中念着不知道水下潜艇编队的亚吉安好友怎么样了，“如果要以一名军人的牺牲来奠定胜利了，那么就由我来好了，让我再给他们的壁炉里加把柴。”

克拉斯克拉了拉军帽，目光锐利。

当克拉斯克边撤边打之时，中部由李由所带领的大批战机编队上原本淡淡的微光变得更加得明亮，因而对aldnoal体系的干扰和迟滞、传输阻断更加得明显。

“〖哼，一群杂虫。〗”

巴鲁克鲁斯往后瞟了一眼那一群群地球联邦的战机编队，此刻，薛克截下的大部分战机和扈从机甲编队已经在地球联邦的炮火过后重新地与敌人缠上。

那群“耍小把戏”的地球联邦阵形战机群已经被领长机甲和扈从机甲团队前后包夹，待到领长机甲向前处理掉装甲空母，便是后面的地球联邦战机死期。

当斐柯伦等人对克拉斯克所在的空舰编队威胁益重之时，原本与斐柯伦等人纠缠的埃克里和星的沃尔洛德精英机甲小队也不得不进入掩护克拉斯克旗舰的守势。

“〖出力不足！〗”拉斐亚在通信中叫 了一句。

所谓的出力不足，并非真的不足，而是在对面的残存四艘空舰撤走前击杀对方的攻击预估输出不够。

“〖没有其它力量么！？〗”巴鲁克鲁斯同样感到惊讶，他没想到后面那些负隅顽抗的战机阵形影响的aldnoal自由空间能场能有这样“差一点”的效果。

“〖那就把登陆城的能量调集过来！〗”驾驶着攻杀型领长机甲的斐柯伦伯爵数次击伤克拉斯克所在的旗舰，却均不能击溃，而他家族所率领的直属领长机甲编队则直接地压制得两支沃尔洛德小队抬不起头。

拉斐亚犹豫了一下，低声道：”〖这不太好吧……〗”

“〖……无妨！击溃敌方，此处前线三十公里区域都将收据于我等轨道骑士之手下，还能够在地球联邦的阵线上打开一个致命的缺口！〗”巴鲁克鲁斯当机立断道。

看着前线斐柯伦伯爵等人乘胜追击，薛克亦不得不把他所指挥的部分阵线向前跟应调动。

此次一战，他并非是作为直接地主力，毕竟军衔与权力摆在了那里，但听取了古－修伊伯爵的建议，他反而直接指挥了大部分的空中阵地阵线，与那三个轨道骑士冲锋杀敌立功的想法完全相左，因而也没有人和他争分属防线的防守指挥权。

一道两道三道，一道道一片片一束束的虹桥跨过十几公里飞落而来，直击还在撤退的克拉斯克装甲空母旗舰侧位，令其激起一阵爆炸与火光。

“〖刘准也男爵！〗”渊戮骑士的支援入场令拉斐亚伯爵等人信心大阵，而刘准也同样在通信中笑道：”〖中部战场上约瑟夫伯爵和敌人打得难解难分，伍斯伯爵、辰月君伯爵与对面的九尾狐纠缠不开，此刻正是我等帝国栋梁为中部战场分担压力，夺取第一个胜利之机！〗”

紧接着，数十道能流的彩虹之桥便划出曲线的轨迹，带着冲击与震颤大气的轰鸣将另一艘还在撤退的地球联邦装甲空母击溃得千疮百孔，庞大的钢铁空母在火焰与爆炸中坠落。

战线后方，如一道梭形天塔的登陆城轰然而动，数之不尽的能量在看不见的空间与轨迹中，向着东部的战场运输而去。

东部另一处，展开的另一座登陆城则激起了大片的气流与风浪，透明的aldnoal场以其为中心辐射向四方，在一望无垠的海洋上铺展开浩大的能量之潮。

东部战场，原本还在李由所率领的战机编队控制下的空间自由能场立刻在如洪水般的能量漩涡冲击下摧枯拉朽地崩溃，如大海倒转洪流般的aldnoal场能量直接覆盖东部战局，而原本在能量输出上总是倍感力有未逮的斐柯伦伯爵等人立刻感到领长机甲的能量充盈。

“轰轰轰轰轰轰！“

随着一道道一片片一重重的能量爆发，前线，薛克的眼中爆发出一片足以将视野填满的能量冲击与爆炸，而那仅剩的两艘装甲空母则在豪雨的攻杀之中，化为一片残骸的金属碎片。

瞬间，东部战场由薇瑟帝国与地球联邦难解难分变为尘埃落定，大片的前线之位，都收入了斐柯伦伯爵等人的控制之下，雷达之上，整个南太平洋东部区域一片透明——

“〖斐柯伦、巴鲁克鲁斯！何动登陆城！！〗”

早已怒得扔掉了贵族礼仪的敬称附加，老将军如雄狮怒吼般的声音在通信中剧烈回荡！
