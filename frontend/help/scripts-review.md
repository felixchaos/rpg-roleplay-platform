# 剧本复核 / KB 复核

## 这页能干嘛

「KB 复核」让你在正式开局之前，检查并修正系统从剧本中自动提取的知识图谱（规范实体、世界线、摄入异常）。**未完成复核的剧本，系统会拒绝开局**——这是防止 GM 因为错误知识产生幻觉的安全锁。

复核入口：剧本管理 → 选中某本剧本 → 右上角「**KB 复核**」。

## 关键概念

### 为什么要复核

LLM 在提取 NPC、地点、势力、世界线等实体时，会根据章节文本做推断。常见问题包括：

- **作者注**混入正文（广告行、更新预告）被误识别为剧情
- **怪章节标题**（卷号、番外序）导致编号跳空
- **同一角色**用多个名字出现，被识别为两个不同实体
- **重要度标错**，核心 NPC 优先级太低，路人 NPC 优先级太高

复核阶段可以手动纠正这些问题，而不需要重新导入整本书。

### 摄入异常标志（ReviewFlags）

复核页顶部会显示几个快速指标：

| 标志 | 含义 |
|---|---|
| 作者非正文 | 识别到的作者注数量（广告/更新预告等） |
| 怪标题 | 章节标题格式异常的数量 |
| 编号缺口 | 章节编号不连续的数量 |
| 广告清洗 | 自动清洗掉的广告行数 |

显示「⚠ 需复核」时建议在开局前至少扫一遍实体表。

### 规范实体表

列出系统识别出的所有角色、地点、组织等实体，包括：名称、类型、首次出现章节、重要度、以及系统生成的摘要。

### 规范世界线

剧本里的多条故事线（主线、支线、番外等）会被识别为独立世界线，并附带各阶段节点列表。

## 常见任务

### 查看摄入异常

打开 KB 复核弹窗后，顶部「摄入正常 / 需复核」旗帜一眼看清整体质量。点开各项数字可以定位具体问题章节。

### 修改实体摘要

1. 在「规范实体」表里找到目标实体，点右侧「**改摘要**」。
2. 输入框变为可编辑状态，修改后点「**存**」保存。
3. GM 会在之后的游戏中使用修正后的摘要。

### 删除误识别实体

如果某个条目明显是广告行或章节序号被错误识别为角色，点「**删**」删除即可。删除后该实体不会再被注入 GM 上下文。

> 删除操作不可撤销。如果误删，重新运行 KB 提取可以恢复（但手动修改会丢失）。

### 章节合并 / 拆分 / 重命名

如果发现章节切分层面的问题（比如两章内容被合并成一章），需要回到「剧本管理 → 查看章节」操作，不在复核弹窗内处理：

- **重命名**：选中章节 → 「重命名」按钮 → 输入新标题。
- **拆分本章**：选中章节 → 「拆分本章」→ 输入从第几个字符处拆分。
- **合并下一章**：选中章节 → 「合并下一章」，两章内容合并为一章。

### 整本重切

如果整本书的切分规则都不对（比如书用英文章节但 auto 识别错了），在「查看章节」弹窗右上角点「**整本重切**」，选新规则后确认，系统会用新规则重新切分整本书。

> 注意：整本重切会覆盖之前所有手动重命名和合并/拆分结果，请谨慎操作。

## 常见问题

**KB 复核完了还是不让开局，什么情况？**
复核通过的条件是「需复核」标志消失（变为「摄入正常」）。如果异常数很多，逐条处理后刷新页面重新检查。

**删掉了一个重要 NPC 实体，怎么恢复？**
在剧本详情的「知识提取」标签重新跑一次提取，系统会重新生成所有实体。但手动修改过的摘要会恢复为 LLM 原始输出。

**世界线显示为空怎么办？**
说明 KB 提取尚未运行，或提取时未识别到多条故事线。先确认已在「知识提取」里完成提取，再重新打开复核页。
