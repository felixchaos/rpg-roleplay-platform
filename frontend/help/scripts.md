# 剧本管理

## 这页能干嘛

「剧本管理」是你上传、整理、分享小说剧本的地方。导入一本 TXT 文件后，系统会自动切分章节、抽取角色卡、生成世界书条目，成为后续 KB 提取和开局的基础。

## 关键概念

### 我的剧本 vs 公开剧本库

- **我的剧本**：只有你自己能看到和编辑，对应「剧本管理」主列表。
- **在线剧本库**：其他用户主动公开分享的剧本。你可以一键「导入到我的剧本」，系统会复制其章节、角色卡和世界书到你自己账户下，不会影响原作者的版本。
- 你的任何一本剧本都可以单独切换公开/私有，互不干扰。

### 章节切分规则（7 种）

导入时系统会先自动嗅探最合适的规则（置信度 ≥ 85% 为绿色）。如果效果不好，可以手动指定：

| 规则 ID | 适用场景 |
|---|---|
| `auto` | 自动识别，优先推荐 |
| `corpus` | 语料章节（带 `第X章` 标头的原始语料格式） |
| `chapter_cn` | 中文章节（「第X章」「第X回」等中文编号） |
| `chapter_en` | 英文章节（Chapter X / CHAPTER X） |
| `number_dot` | 数字点号（`1.`、`2.` 开头） |
| `paren_num` | 括号编号（`（1）`、`【1】`等） |
| `custom` | 自定义正则，手动填写匹配模式 |

导入完成后，如果切分效果不满意，可以在「章节管理」里单章重命名、拆分、合并，或一键「整本重切」。

### 向量索引

切分完成后还需要「建立向量索引」，KB 提取才能检索到相关段落。右上角「更多 → 建立向量索引」启动后台任务，进度会实时显示在按钮文字上（`向量化中 XX%`）。

## 常见任务

### 导入一本新剧本

1. 点击右上角「**导入剧本**」，进入导入全屏页。
2. 上传 `.txt` 文件（UTF-8，建议先去掉广告行）。
3. 系统显示「章节切分预览」弹窗，展示章节数、总字数、置信度和前几章列表。
4. 如果置信度偏低，点「**换规则重试**」切换切分规则，直到预览正常。
5. 确认后点「**确认导入**」，系统在后台完成全量切分。

> 还有另一种方式：「**导入剧本包**」接受 `.zip` 格式的打包文件（含章节、角色卡、世界书），适合把从朋友处拿到的导出包一键还原。

### 从在线剧本库导入他人剧本

切到「在线剧本库」标签页 → 搜索标题/简介 → 点「**导入到我的剧本**」。导入完成后该剧本会出现在「我的剧本」列表里。

### 公开 / 私有切换

在剧本列表点击某本剧本，选中后在下方详情面板的右上角下拉菜单里选「**公开分享到剧本库**」。再次点击「取消公开分享」即可恢复私有。公开时系统会弹确认框，告知其他用户可以浏览并导入你的章节 / 角色卡 / 世界书。

### 编辑剧本元信息

选中剧本 → 详情面板 → **概览** 标签：可看到标题、章节数、字数、切分模式、置信度、存档数等信息。如果需要调整 GM 行为参数（不限于标题），切到「**参数**」标签，点「编辑覆盖设定」打开 JSON 编辑器，直接修改 `script_overrides` 字段后保存。

> 注意：标题和简介目前通过覆盖设定 JSON 的对应字段修改，不是独立的表单字段。

### 查看章节列表

详情面板右上角点「**查看章节**」（或行操作下拉选「查看章节 / 重切分」），打开章节管理弹窗。左侧章节列表，右侧显示当前章节正文前 4000 字。

## 常见问题

**导入后章节数不对，怎么办？**
打开「查看章节」→「整本重切」，换一个切分规则试试。切分规则优先推荐 `auto`，如果识别为中文章节标题但结果仍然乱，可以手动选 `chapter_cn` 或 `custom` 并填写正则。

**导入完但始终看不到 NPC 角色卡？**
先「建立向量索引」，再到详情面板「知识提取」标签运行 KB 提取。提取结束后切到「NPC 角色卡」标签即可看到结果。

**置信度低于 70% 是什么意思？**
说明系统识别章节标题的把握很小，实际切分可能错位。建议在切分预览里检查章节列表，手动选规则或使用「整本重切」修正。

**删除剧本会影响什么？**
删除剧本时，对应的所有存档、向量索引也会一并清理。操作前请确认。
